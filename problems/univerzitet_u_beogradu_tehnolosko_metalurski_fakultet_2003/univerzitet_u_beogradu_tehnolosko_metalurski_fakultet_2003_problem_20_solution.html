Najveći prirodan parametar \(p\) za striktnu racionalnu nejednakost

Algebra — diskriminanta kvadratnog trinoma kao alat za uniformne nejednakosti

📋 Tekst zadatka

Najveći prirodan broj \(p\), takav da nejednakost

\[ \left| \dfrac{x^2 + p x + 1}{x^2 + x + 1} \right| < 2 \]

važi za svako \(x \in \mathbb{R}\), je:

(A) \(2\)
(B) \(6\)
(C) \(8\)
(D) \(3\)
(E) \(4\)

🎯 Plan rešavanja

Pokazaćemo da je imenilac \(x^2 + x + 1\) strogo pozitivan, pa apsolutnu vrednost možemo „raspakovati" u dvostruku nejednakost. Svaka strana daje uslov da kvadratni trinom bude pozitivan za svako \(x\), što u oba slučaja znači negativna diskriminanta. Presek dobijenih intervala za \(p\) daje konačan skup, iz koga čitamo najveći prirodan broj.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Kvadratni trinom \(ax^2 + bx + c\) sa \(a > 0\) je pozitivan za svako \(x \in \mathbb{R}\) ako i samo ako je diskriminanta \(D = b^2 - 4ac < 0\) (parabola u celosti iznad \(x\)-ose).
  • Apsolutna vrednost: \(|A| < B \;\Leftrightarrow\; -B < A < B\) (uz \(B > 0\)).
  • Ako je imenilac strogo pozitivan, onda \(\left|\frac{A}{B}\right| < k \;\Leftrightarrow\; |A| < k\,B\).
  • Imenilac \(x^2 + x + 1\) ima diskriminantu \(1 - 4 = -3 < 0\), pa je strogo pozitivan za svako \(x\).

📐 Vizuelni prikaz

Grafik funkcije \(f(x) = (x^2 + p x + 1)/(x^2 + x + 1)\) za tri vrednosti parametra \(p\). Plave horizontale \(y = \pm 2\) su zabranjene granice. Krive za \(p = 3\) i \(p = 0\) ostaju strogo unutar pojasa, dok kriva za \(p = 4\) tačno dotiče gornju granicu \(y = 2\) (u tački \(x = 1\)) — pa nejednakost ne važi strogo.

✏️ Rešenje korak po korak

1

Pokažimo da je imenilac strogo pozitivan

Posmatrajmo trinom u imeniocu i izračunajmo njegovu diskriminantu:

\[ x^2 + x + 1, \qquad D = 1^2 - 4\cdot 1 \cdot 1 = -3 < 0. \]

Pošto je vodeći koeficijent \(1 > 0\) a diskriminanta negativna, parabola \(y = x^2 + x + 1\) je u celosti iznad \(x\)-ose, pa je \(x^2 + x + 1 > 0\) za svako \(x \in \mathbb{R}\).

Proveri sebe: zašto je ovaj korak ključan?

Bez ovog opažanja ne bismo smeli da pomnožimo nejednakost imeniocem (jer pri množenju negativnim brojem znak nejednakosti se okreće). Strogo pozitivan imenilac dozvoljava da uklonimo apsolutnu vrednost imenioca: \(|x^2+x+1| = x^2+x+1\).

2

Razložimo apsolutnu vrednost u dvostruku nejednakost

Pošto je imenilac pozitivan, polazna nejednakost ekvivalentna je sa

\[ |x^2 + p x + 1| < 2(x^2 + x + 1). \]

Pravilo \(|A| < B \Leftrightarrow -B < A < B\) (uz \(B > 0\)) daje dva uslova istovremeno:

\[ -2(x^2 + x + 1) \;<\; x^2 + p x + 1 \;<\; 2(x^2 + x + 1). \]

Sada ćemo svaku nejednakost svesti na zahtev „kvadratni trinom je strogo pozitivan za svako \(x\)".

3

Gornja nejednakost — uslov \(D_1 < 0\)

Sredimo \(x^2 + p x + 1 < 2(x^2 + x + 1)\):

\[ x^2 + p x + 1 < 2x^2 + 2x + 2 \quad\Longleftrightarrow\quad 0 < x^2 + (2 - p)\,x + 1. \]

Levi vodeći koeficijent (\(=1\)) je pozitivan, pa trinom \(x^2 + (2-p)x + 1\) je strogo pozitivan za svako \(x\) ako i samo ako je njegova diskriminanta strogo negativna:

\[ D_1 = (2 - p)^2 - 4 < 0. \]
\[ (2 - p)^2 < 4 \;\Longleftrightarrow\; |2 - p| < 2 \;\Longleftrightarrow\; -2 < 2 - p < 2. \]

Oduzimanjem 2, a zatim množenjem sa \(-1\) (što okreće smer nejednakosti):

\[ \boxed{\,0 < p < 4\,}. \]
4

Donja nejednakost — uslov \(D_2 < 0\)

Sredimo \(-2(x^2 + x + 1) < x^2 + p x + 1\):

\[ -2x^2 - 2x - 2 < x^2 + p x + 1 \quad\Longleftrightarrow\quad 0 < 3x^2 + (p + 2)\,x + 3. \]

Vodeći koeficijent je \(3 > 0\). Diskriminanta mora biti negativna:

\[ D_2 = (p + 2)^2 - 4\cdot 3 \cdot 3 = (p + 2)^2 - 36 < 0. \]
\[ (p + 2)^2 < 36 \;\Longleftrightarrow\; |p + 2| < 6 \;\Longleftrightarrow\; -6 < p + 2 < 6. \]

Oduzimanjem 2:

\[ \boxed{\,-8 < p < 4\,}. \]
Proveri sebe: zašto je ovde \(a = 3\), a ne \(1\)?

Sabiranjem \(-2x^2\) (sa leve strane) i \(x^2\) (sa desne strane), dobijamo \(3x^2\) na desnoj strani. Diskriminanta zato uzima \(a = 3, b = p+2, c = 3\), tj. \(D_2 = b^2 - 4ac = (p+2)^2 - 36\).

5

Presek uslova i odabir najvećeg prirodnog broja

Oba uslova moraju biti istovremeno zadovoljena, pa pravimo presek intervala:

\[ p \in (0, 4) \,\cap\, (-8, 4) \;=\; (0, 4). \]

Prirodni brojevi iz intervala \((0, 4)\) su \(1, 2, 3\) (4 nije uključen, jer je granica striktna). Najveći među njima je:

\[ p_{\max} = 3. \]
6

Provera — zašto \(p = 3\) prolazi, a \(p = 4\) ne

Za \(p = 3\):

\[ D_1 = (2-3)^2 - 4 = 1 - 4 = -3 < 0, \qquad D_2 = (3+2)^2 - 36 = 25 - 36 = -11 < 0. \]

Obe diskriminante su negativne — nejednakost važi za svako \(x\). \(\checkmark\)

Za \(p = 4\): u koraku 3 dobijamo \(D_1 = (2-4)^2 - 4 = 0\), pa je trinom \(x^2 - 2x + 1 = (x-1)^2 \geq 0\), sa nulom u \(x = 1\). Tu se gornja nejednakost dotiče:

\[ \text{za } x = 1:\quad \dfrac{1^2 + 4\cdot 1 + 1}{1^2 + 1 + 1} = \dfrac{6}{3} = 2, \]

što ne zadovoljava strogo \(< 2\). Dakle \(p = 4\) ne prolazi, a sve veće prirodne vrednosti (\(6, 8\)) tim pre ne mogu biti rešenje.

Proveri sebe: numerička kontrola za \(p = 3\)

U \(x = 1\): \(\dfrac{1 + 3 + 1}{1 + 1 + 1} = \dfrac{5}{3} \approx 1{,}67 < 2\). \(\checkmark\)

U \(x = -1\): \(\dfrac{1 - 3 + 1}{1 - 1 + 1} = \dfrac{-1}{1} = -1\), \(|-1| = 1 < 2\). \(\checkmark\)

Limesi pri \(x \to \pm\infty\) idu ka \(1\) (odnos vodećih koeficijenata), takođe unutar pojasa.

💡 Ključni uvid

Strogo pozitivan imenilac (provera diskriminante) pretvara naizgled složenu racionalnu nejednakost sa apsolutnom vrednošću u par standardnih kvadratnih nejednakosti. Tu je „motor" rešenja: kvadratni trinom \(ax^2 + bx + c\) sa \(a>0\) je pozitivan na celom \(\mathbb{R}\) tačno kada je \(D < 0\). Dva uslova daju dva intervala za \(p\), a presek je gotov rezultat.

✅ Konačan odgovor

\[ p_{\max} = 3 \]

(A) \(2\) (B) \(6\) (C) \(8\) (D) \(3\) (E) \(4\)

Provera: za \(p = 3\) obe diskriminante \((2-p)^2 - 4 = -3\) i \((p+2)^2 - 36 = -11\) su negativne, pa nejednakost važi za svako \(x\). Za \(p = 4\) gornja granica se dostiže pri \(x = 1\): \(\dfrac{1 + 4 + 1}{1 + 1 + 1} = \dfrac{6}{3} = 2\), pa nejednakost nije strogo zadovoljena. Dakle najveći prirodan broj jeste \(p = 3\).

⚠️ Česte greške

  • Zaborav apsolutne vrednosti: Mnogi rešavaju samo \(\dfrac{x^2+px+1}{x^2+x+1} < 2\), a propuste donji uslov \(\dfrac{x^2+px+1}{x^2+x+1} > -2\). Posledica: dobijaju širi interval \(p \in (-8, 4)\) i biraju \(p = 4\) ili veće — netačno.
  • Nedovoljno strog uslov: Pisanje \(D \le 0\) umesto \(D < 0\). Kada je \(D = 0\), trinom dotiče \(x\)-osu (jedna realna nula), pa nejednakost \(>0\) ne važi u toj tački. Zato granica \(p = 4\) otpada.
  • Provera za \(p = 4\) preko vodećih koeficijenata limesa: Za \(x \to \pm \infty\) odnos teži \(1\), pa neki misle „interval u redu". Ali nejednakost mora važiti za svako \(x\), uključujući kritičnu tačku \(x = 1\), gde upravo otkažuje za \(p = 4\).
  • Greška sa znakovima pri prevrtanju: U koraku 3, prelazak od \(-2 < 2 - p < 2\) ka \(0 < p < 4\) zahteva oduzimanje 2 i množenje sa \(-1\), što obrće znake nejednakosti.

🏆 Dodatni izazov

Generalizuj zadatak: za koje vrednosti pozitivne konstante \(k\) važi nejednakost

\[ \left| \dfrac{x^2 + p x + 1}{x^2 + x + 1} \right| < k \quad \text{za svako } x \in \mathbb{R}, \]

i koji je tada najveći prirodan \(p\) u zavisnosti od \(k\)?

Pogledaj nagoveštaj

Isti postupak vodi do dve nejednakosti:

\[ (k-1)x^2 + (k - p)x + (k-1) > 0, \qquad (k+1)x^2 + (p+k)x + (k+1) > 0. \]

(Originalni zadatak je slučaj \(k = 2\).) Za \(k > 1\), oba vodeća koeficijenta su pozitivna i diskriminante daju:

\[ (k - p)^2 < 4(k-1)^2, \qquad (p + k)^2 < 4(k+1)^2. \]

Iz prve: \(2 - k < p < 3k - 2\). Iz druge: \(-3k - 2 < p < k + 2\).

Presek (pretpostavljajući \(k > 1\)): \(2 - k < p < k + 2\), pa je najveći prirodan \(p = \lceil k + 2 \rceil - 1\) (najveći prirodan strogo manji od \(k+2\)).

Za \(k = 2\): interval \((0, 4)\), najveći prirodan \(p = 3\) — saglasno sa originalnim rešenjem.
