Realnost rešenja i uslov \(x_1^2 + x_2^2 - x_1 x_2 > 0\)

Algebra — kvadratna jednačina sa parametrom, Vijetove formule, diskriminanta

📋 Tekst zadatka

Data je jednačina \(x^2 + (a+1)x + a^2 - 1 = 0\), \(a \in \mathbb{R}\). Rešenja \(x_1\) i \(x_2\) jednačine su realna i zadovoljavaju uslov \(x_1^2 + x_2^2 - x_1 x_2 > 0\) ako i samo ako je:

(A) \(-1 < a < 2\)
(B) \(-1 \leq a < 2\)
(C) \(-1 < a \leq \dfrac{5}{3}\)
(D) \(|a| \geq 2\)
(E) \(|a| < 2\)

🎯 Plan rešavanja

Postavljamo dva uslova: prvo \(D \geq 0\) za realnost rešenja, zatim transformišemo \(x_1^2 + x_2^2 - x_1 x_2\) preko Vijetovih formula u izraz po \(a\) i tražimo da je veći od nule. Konačni skup je presek dva intervala.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Vijetove formule za \(x^2 + px + q = 0\) sa rešenjima \(x_1, x_2\): \(x_1 + x_2 = -p\), \(x_1 x_2 = q\).
  • Diskriminanta: \(D = p^2 - 4q\) (za moničku kvadratnu). Realna rešenja \(\Leftrightarrow D \geq 0\).
  • Identitet sa simetričnim funkcijama: \(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2 x_1 x_2\), pa \(x_1^2 + x_2^2 - x_1 x_2 = (x_1 + x_2)^2 - 3 x_1 x_2\).
  • Kvadratna nejednačina: \(\alpha x^2 + \beta x + \gamma < 0\) sa pozitivnim \(\alpha\) ima rešenje između nula trinoma.

📐 Vizuelni prikaz

Brojevna prava: skup koji daje realna rešenja \([-1,\,5/3]\) i skup u kojem je uslov \(x_1^2+x_2^2-x_1 x_2 > 0\) ispunjen \((-1,\,2)\). Zelena traka je njihov presek \((-1,\,5/3]\).

✏️ Rešenje korak po korak

1

Uslov 1: realnost rešenja (\(D \geq 0\))

Za \(x^2 + (a+1)x + (a^2 - 1) = 0\) diskriminanta je:

\[ D = (a+1)^2 - 4(a^2 - 1) \]
\[ = a^2 + 2a + 1 - 4a^2 + 4 = -3a^2 + 2a + 5 \]

Tražimo \(D \geq 0\):

\[ -3a^2 + 2a + 5 \geq 0 \;\Longleftrightarrow\; 3a^2 - 2a - 5 \leq 0 \]
2

Rešavanje \(3a^2 - 2a - 5 \leq 0\)

Nule trinoma:

\[ a = \dfrac{2 \pm \sqrt{4 + 60}}{6} = \dfrac{2 \pm 8}{6} \;\Longrightarrow\; a = -1 \text{ ili } a = \dfrac{5}{3} \]

Pošto je vodeći koeficijent pozitivan, trinom je \(\leq 0\) između nula:

\[ \boxed{\,-1 \leq a \leq \dfrac{5}{3}\,} \quad (\text{Uslov 1}) \]
3

Uslov 2: \(x_1^2 + x_2^2 - x_1 x_2 > 0\) preko Vijetovih formula

Po Vijetovim formulama: \(x_1 + x_2 = -(a + 1)\) i \(x_1 x_2 = a^2 - 1\). Koristimo identitet:

\[ x_1^2 + x_2^2 - x_1 x_2 = (x_1 + x_2)^2 - 3 x_1 x_2 \]

Zamenom:

\[ = (a+1)^2 - 3(a^2 - 1) = a^2 + 2a + 1 - 3a^2 + 3 = -2a^2 + 2a + 4 \]
Proveri sebe: zašto \(x_1^2 + x_2^2 - x_1 x_2 = (x_1+x_2)^2 - 3 x_1 x_2\)?

\((x_1 + x_2)^2 = x_1^2 + 2 x_1 x_2 + x_2^2\). Otuda \(x_1^2 + x_2^2 = (x_1+x_2)^2 - 2 x_1 x_2\). Dodavanjem \(- x_1 x_2\): \(x_1^2 + x_2^2 - x_1 x_2 = (x_1+x_2)^2 - 2 x_1 x_2 - x_1 x_2 = (x_1+x_2)^2 - 3 x_1 x_2\). ✓

4

Rešavanje \(-2a^2 + 2a + 4 > 0\)

Delimo sa \(-2\) (znak nejednakosti se menja):

\[ a^2 - a - 2 < 0 \]

Faktorizacija: \(a^2 - a - 2 = (a - 2)(a + 1)\), pa:

\[ (a - 2)(a + 1) < 0 \;\Longleftrightarrow\; \boxed{\,-1 < a < 2\,} \quad (\text{Uslov 2}) \]

(Strogo nejednakost — krajevi \(-1\) i \(2\) nisu uključeni.)

5

Presek uslova

Tražimo \(a\) koje istovremeno zadovoljava oba uslova:

\[ \underbrace{[-1,\,\tfrac{5}{3}]}_{\text{Uslov 1}} \;\cap\; \underbrace{(-1,\,2)}_{\text{Uslov 2}} \;=\; \left(-1,\,\dfrac{5}{3}\right] \]

Granica \(a = -1\) se isključuje (Uslov 2 je strog), a \(a = \dfrac{5}{3}\) se uključuje (Uslov 1 dozvoljava jednakost).

Proveri sebe: šta se dešava na granicama \(a = -1\) i \(a = 5/3\)?

\(a = -1\): jednačina postaje \(x^2 + 0 \cdot x + 0 = 0\), tj. \(x^2 = 0\), pa \(x_1 = x_2 = 0\). Tada \(x_1^2 + x_2^2 - x_1 x_2 = 0\), ne \(> 0\). Isključujemo.

\(a = 5/3\): \(D = 0\), pa imamo dvostruki koren \(x_1 = x_2 = -\dfrac{4}{3}\). Tada \(x_1^2 + x_2^2 - x_1 x_2 = 2 \cdot \dfrac{16}{9} - \dfrac{16}{9} = \dfrac{16}{9} > 0\). ✓ Uključujemo.

💡 Ključni uvid

Simetrični izrazi u \(x_1\) i \(x_2\) (kao što je \(x_1^2 + x_2^2 - x_1 x_2\)) uvek se mogu izraziti samo preko zbira i proizvoda korenova. Zato Vijetove formule pretvaraju problem u kvadratnu nejednačinu po parametru \(a\) — bez eksplicitnog rešavanja jednačine. Završetak je strogo gledanje granica: koju koja nejednakost dopušta jednakost a koja ne.

✅ Konačan odgovor

\[ -1 < a \leq \dfrac{5}{3} \]

(A) \(-1 < a < 2\) (B) \(-1 \leq a < 2\) (C) \(-1 < a \leq \dfrac{5}{3}\) (D) \(|a| \geq 2\) (E) \(|a| < 2\)

Provera (granice): za \(a = 0\) (unutrašnja tačka): \(D = 5 > 0\), a vrednost izraza je \(4 > 0\). Za \(a = -1\): rešenje \(0\), izraz \(= 0\) (isključeno). Za \(a = \dfrac{5}{3}\): dvostruki koren \(-\dfrac{4}{3}\), izraz \(= \dfrac{16}{9} > 0\) (uključeno). ✓

⚠️ Česte greške

  • Zaboravljanje uslova realnosti: Ako se zahteva da rešenja postoje, mora se prvo postaviti \(D \geq 0\). Bez toga bi odgovor bio (A) — pogrešno.
  • Pogrešan smer nejednakosti pri deljenju negativnim brojem: Pri prelazu \(-2a^2 + 2a + 4 > 0 \to a^2 - a - 2 < 0\) znak se obrće. Ako se ovo zaboravi, dobija se \((-\infty, -1) \cup (2, \infty)\), što vodi ka odgovoru (D).
  • Brkanje strogih i nestrogih nejednakosti na granicama: \(D \geq 0\) (sa jednakošću) ali izraz \(> 0\) (strog). Zato leva granica \(a = -1\) ispada, a desna \(a = \dfrac{5}{3}\) ostaje.
  • Pokušaj rešavanja eksplicitnih korena: Eksplicitno računanje \(x_{1,2}\) preko \(\sqrt{D}\) je dugotrajno. Vijetove formule rešavaju problem u par redova.

🏆 Dodatni izazov

Neka su \(x_1, x_2\) realna rešenja iste jednačine \(x^2 + (a+1)x + a^2 - 1 = 0\). Za koje vrednosti parametra \(a\) važi \(x_1^2 + x_2^2 = 5\)?

Pogledaj nagoveštaj

\(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2 x_1 x_2 = (a+1)^2 - 2(a^2 - 1) = -a^2 + 2a + 3\).

Postavljamo \(-a^2 + 2a + 3 = 5\), tj. \(a^2 - 2a + 2 = 0\). Diskriminanta ove kvadratne po \(a\) je \(4 - 8 = -4 < 0\) — nema realnih \(a\). Dakle, ne postoji \(a \in \mathbb{R}\) za koji bi \(x_1^2 + x_2^2 = 5\). (Maksimalna vrednost \(-a^2 + 2a + 3\) na intervalu \([-1, 5/3]\) postiže se u \(a = 1\) i iznosi \(4\) — uvek manja od 5.)
