Nula-skup proizvoda dveju funkcija

Teorija skupova i funkcije — veza između nula-skupova

Tekst zadatka

Neka su \( f, g: \mathbb{R} \to \mathbb{R} \) funkcije i neka je \( A \) skup svih realnih brojeva \( x \) za koje je \( f(x) = 0 \), \( B \) skup svih realnih brojeva \( x \) za koje je \( g(x) = 0 \) i \( C \) skup svih realnih brojeva \( x \) za koje je \( f(x)g(x) = 0 \). Tada uvek važi:

(A) \( C = A \cap B \)
(B) \( C = A \cup B \)
(C) \( C = A \setminus B \)
(D) \( C = B \setminus A \)
(E) \( C = A \times B \)

Plan rešavanja

Primenićemo osnovno svojstvo realnih brojeva — da je proizvod jednak nuli ako i samo ako je bar jedan činilac jednak nuli — i prevesti ga na jezik teorije skupova.

Podsetnik iz teorije

Ključne formule i teoreme
  • Nula proizvoda Za realne brojeve: \( a \cdot b = 0 \iff a = 0 \text{ ili } b = 0 \)
  • Unija skupova \( A \cup B = \{x : x \in A \text{ ili } x \in B\} \)
  • Presek skupova \( A \cap B = \{x : x \in A \text{ i } x \in B\} \)

Vizuelni prikaz

Venov dijagram: skup \( C = A \cup B \) obuhvata sve tačke gde je \( f(x) = 0 \) ili \( g(x) = 0 \)

Rešenje korak po korak

1

Definišemo skupove precizno

Prema uslovima zadatka:

\[ A = \{x \in \mathbb{R} : f(x) = 0\} \]
\[ B = \{x \in \mathbb{R} : g(x) = 0\} \]
\[ C = \{x \in \mathbb{R} : f(x) \cdot g(x) = 0\} \]
2

Primenjujemo svojstvo nule proizvoda

Za realne brojeve važi osnovno algebarsko svojstvo:

\[ f(x) \cdot g(x) = 0 \iff f(x) = 0 \text{ ili } g(x) = 0 \]

Ovo znači da je \( x \in C \) ako i samo ako je \( x \in A \) ili \( x \in B \).

Proveri sebe: Zašto ne važi „i" umesto „ili"?

Dovoljno je da samo jedan faktor bude nula da bi proizvod bio nula. Na primer, \( 0 \cdot 5 = 0 \), iako je drugi činilac različit od nule.

3

Prevodimo na jezik teorije skupova

Po definiciji unije skupova:

\[ A \cup B = \{x : x \in A \text{ ili } x \in B\} \]

Pošto je \( x \in C \iff x \in A \text{ ili } x \in B \iff x \in A \cup B \), zaključujemo:

\[ C = A \cup B \]
Proveri sebe: Zašto opcija \( C = A \cap B \) nije tačna?

Presek \( A \cap B \) sadrži samo one \( x \) za koje je \( f(x) = 0 \) i \( g(x) = 0 \) istovremeno. Ali za \( f(x)g(x) = 0 \) dovoljno je da samo jedna funkcija bude nula, pa \( A \cap B \subseteq C \), ali generalno \( A \cap B \neq C \).

4

Eliminišemo ostale opcije

Konkretan primer: neka je \( f(x) = x \) i \( g(x) = x - 1 \). Tada:

\[ A = \{0\}, \quad B = \{1\}, \quad C = \{0, 1\} \]
  • \( A \cap B = \emptyset \neq C \) — opcija (A) ne važi
  • \( A \cup B = \{0, 1\} = C \) — opcija (B) važi \( \checkmark \)
  • \( A \setminus B = \{0\} \neq C \) — opcija (C) ne važi
  • \( B \setminus A = \{1\} \neq C \) — opcija (D) ne važi
  • \( A \times B = \{(0, 1)\} \) — ovo je skup uređenih parova, ne podskup od \( \mathbb{R} \) — opcija (E) nema smisla

Ključni uvid

Ovo je zadatak koji povezuje algebru i teoriju skupova. Ključno svojstvo realnih brojeva — da je proizvod nula akko je bar jedan činilac nula — direktno se preslikava u operaciju unije skupova nula. Logička veza „ili" u algebri odgovara operaciji \( \cup \) u teoriji skupova.

Konačan odgovor

\[ C = A \cup B \]

(A) \( C = A \cap B \) (B) \( C = A \cup B \) (C) \( C = A \setminus B \) (D) \( C = B \setminus A \) (E) \( C = A \times B \)

Verifikacija: Za \( f(x) = x \), \( g(x) = x - 1 \): \( f(x)g(x) = x(x-1) = 0 \) daje \( x = 0 \) ili \( x = 1 \), što je \( A \cup B = \{0\} \cup \{1\} = \{0, 1\} = C \).

Česte greške

  • Mešanje „ili" i „i": \( f(x)g(x) = 0 \) zahteva da je bar jedan činilac nula (ili), a ne oba (i). Zato je odgovor \( \cup \), a ne \( \cap \).
  • Dekartov proizvod: \( A \times B \) daje skup uređenih parova, koji nije podskup od \( \mathbb{R} \). Ova opcija nema smisla u kontekstu zadatka.
  • Razlika skupova: \( A \setminus B \) ili \( B \setminus A \) isključuju deo rešenja, pa ne mogu biti tačan odgovor.

Dodatni izazov

Ako umesto proizvoda posmatramo skup \( D = \{x \in \mathbb{R} : f(x)^2 + g(x)^2 = 0\} \), koja operacija nad skupovima \( A \) i \( B \) daje skup \( D \)?

Pogledaj nagoveštaj

Pošto su \( f(x)^2 \geq 0 \) i \( g(x)^2 \geq 0 \), njihov zbir je nula ako i samo ako su oba jednaka nuli: \( f(x) = 0 \) i \( g(x) = 0 \). Dakle \( D = A \cap B \) — presek skupova!
