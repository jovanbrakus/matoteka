Jednačina \( \sin^4 x = 1 - \cos^4 x \) na \([0,2\pi]\)

Trigonometrija — identiteti, dvostruki ugao, rešenja u datom intervalu

Tekst zadatka

Sva rešenja jednačine \( \sin^4 x = 1 - \cos^4 x \) koja pripadaju intervalu \( [0, 2\pi] \) su:

(A) \( \dfrac{\pi}{2},\ \dfrac{3\pi}{2} \)
(B) \( 0,\ \dfrac{\pi}{2},\ \pi,\ \dfrac{3\pi}{2} \)
(C) \( 0,\ \dfrac{\pi}{2},\ \pi,\ \dfrac{3\pi}{2},\ 2\pi \)
(D) \( 0,\ \pi,\ 2\pi \)

Plan rešavanja

Jednačinu prebacimo u oblik \( \sin^4 x + \cos^4 x = 1 \), pa kvadriranjem osnovnog identiteta \( \sin^2 x + \cos^2 x = 1 \) dobijemo \( \sin^4 x + \cos^4 x = 1 - 2\sin^2 x\cos^2 x \). Odavde sledi \( \sin x \cos x = 0 \), tj. \( \sin(2x) = 0 \), pa brojimo rešenja u \([0, 2\pi]\).

Podsetnik iz teorije

Ključni identiteti
  • Osnovni identitet: \( \sin^2 x + \cos^2 x = 1 \) za svako \( x \in \mathbb{R} \).
  • Kvadrat zbira: \( (a+b)^2 = a^2 + 2ab + b^2 \), pa je \( (\sin^2 x + \cos^2 x)^2 = \sin^4 x + 2\sin^2 x\cos^2 x + \cos^4 x \).
  • Dvostruki ugao: \( \sin(2x) = 2\sin x \cos x \).
  • Nule sinusa: \( \sin t = 0 \iff t = k\pi,\ k \in \mathbb{Z} \).

Vizuelni prikaz

Grafik funkcije \( y = \sin^4 x + \cos^4 x \) na \([0, 2\pi]\) i horizontalna prava \( y = 1 \). Preseci su tačno tražena rešenja.

Funkcija \( \sin^4 x + \cos^4 x \) oscilira između \( \tfrac{1}{2} \) (u tačkama \(x = \pi/4 + k\pi/2\)) i \( 1 \) (u tačkama \(x = k\pi/2\)).

Rešenje korak po korak

1

Prebacivanje na jednu stranu

Jednačina \( \sin^4 x = 1 - \cos^4 x \) ekvivalentna je sa:

\[ \sin^4 x + \cos^4 x = 1. \]
2

Kvadriranje osnovnog identiteta

Kvadriranjem \( \sin^2 x + \cos^2 x = 1 \) dobijamo:

\[ (\sin^2 x + \cos^2 x)^2 = \sin^4 x + 2\sin^2 x \cos^2 x + \cos^4 x = 1. \]

Odavde sledi ključna veza:

\[ \sin^4 x + \cos^4 x = 1 - 2\sin^2 x \cos^2 x. \]
Proveri sebe: čemu je jednako \( \sin^4 x + \cos^4 x \) za \( x = \pi/4 \)?

\( \sin(\pi/4) = \cos(\pi/4) = \tfrac{\sqrt{2}}{2} \), pa je \( \sin^4 + \cos^4 = 2\cdot \tfrac{1}{4} = \tfrac{1}{2} \). Takođe \( 1 - 2\cdot \tfrac{1}{2}\cdot \tfrac{1}{2} = 1 - \tfrac{1}{2} = \tfrac{1}{2} \). Formula se slaže.

3

Svođenje na \( \sin(2x) = 0 \)

Kombinujući \( \sin^4 x + \cos^4 x = 1 \) sa prethodnom jednakošću:

\[ 1 - 2\sin^2 x \cos^2 x = 1 \quad \Rightarrow \quad \sin^2 x \cos^2 x = 0. \]

Odavde je \( \sin x \cos x = 0 \). Koristimo dvostruki ugao \( \sin(2x) = 2\sin x \cos x \):

\[ \tfrac{1}{2}\sin(2x) = 0 \quad \Rightarrow \quad \sin(2x) = 0. \]
Proveri sebe: zašto iz \( \sin^2 x \cos^2 x = 0 \) sledi \( \sin x \cos x = 0 \)?

\( \sin^2 x \cos^2 x = (\sin x \cos x)^2 \). Ako je kvadrat realnog broja nula, onda je i sam broj nula.

4

Sva rešenja u \([0, 2\pi]\)

\( \sin(2x) = 0 \iff 2x = k\pi \iff x = \dfrac{k\pi}{2},\ k \in \mathbb{Z} \). Tražimo \( x \in [0, 2\pi] \):

\[ \begin{array}{c|c} k & x = k\pi/2 \\ \hline 0 & 0 \\ 1 & \pi/2 \\ 2 & \pi \\ 3 & 3\pi/2 \\ 4 & 2\pi \end{array} \]

Dobijamo \(5\) rešenja: \( 0,\ \dfrac{\pi}{2},\ \pi,\ \dfrac{3\pi}{2},\ 2\pi \).

Proveri sebe: zašto i krajevi \(0\) i \(2\pi\) ulaze u rešenja?

Interval \( [0, 2\pi] \) je zatvoren — uključuje oba kraja. U \(x = 0\) je \( \sin^4 0 + \cos^4 0 = 0 + 1 = 1 \), i isto u \(x = 2\pi\). Oba zadovoljavaju jednačinu.

Ključni uvid

Izrazi tipa \( \sin^4 x + \cos^4 x \) retko se obrađuju direktno — skoro uvek se koristi trik kvadriranja osnovnog identiteta \( \sin^2 + \cos^2 = 1 \). Tako se četvrti stepeni prevode u drugi preko \( 2\sin^2 x\cos^2 x \), koji je dalje povezan sa dvostrukim uglom \( \sin(2x) \).

Konačan odgovor

\[ x \in \left\{ 0,\ \tfrac{\pi}{2},\ \pi,\ \tfrac{3\pi}{2},\ 2\pi \right\} \]

(A) \( \tfrac{\pi}{2},\ \tfrac{3\pi}{2} \) (B) \( 0,\ \tfrac{\pi}{2},\ \pi,\ \tfrac{3\pi}{2} \) (C) \( 0,\ \tfrac{\pi}{2},\ \pi,\ \tfrac{3\pi}{2},\ 2\pi \) (D) \( 0,\ \pi,\ 2\pi \)

Provera: \( x = 2\pi \) daje \( \sin^4(2\pi) + \cos^4(2\pi) = 0 + 1 = 1 \) — zadovoljava. Dakle, tačan odgovor je (C).

Česte greške

  • Ispuštanje krajeva intervala: Interval \([0, 2\pi]\) je zatvoren, pa i \( x = 0 \) i \( x = 2\pi \) ulaze u rešenja. Odgovor (B) ispušta \( 2\pi \), odgovor (D) ispušta \( \pi/2 \) i \( 3\pi/2 \).
  • Direktno faktorisanje \( \sin^4 + \cos^4 \) kao zbir kvadrata: Nije moguće realno faktorisati \( a^2 + b^2 \); potrebno je kvadrirati identitet \( a + b = 1 \) sa \( a = \sin^2 x \), \( b = \cos^2 x \).
  • Skraćivanje sa \( \sin x \) ili \( \cos x \): Iz \( \sin x \cos x = 0 \) sledi \( \sin x = 0 \) ili \( \cos x = 0 \), a ne samo jedan od slučajeva. Uzimanje samo jednog gubi pola rešenja.

Dodatni izazov

Reši jednačinu \( \sin^6 x + \cos^6 x = 1 \) u \( [0, 2\pi] \). Da li dobijaš istu kolekciju rešenja kao u ovom zadatku?

Pogledaj nagoveštaj

Koristi identitet \( a^3 + b^3 = (a+b)(a^2 - ab + b^2) \) sa \( a = \sin^2 x \), \( b = \cos^2 x \). Dobija se \( \sin^6 x + \cos^6 x = 1 - 3\sin^2 x \cos^2 x \). Jednakost sa \(1\) ponovo daje \( \sin x \cos x = 0 \), pa su rešenja ista: \( 0,\ \tfrac{\pi}{2},\ \pi,\ \tfrac{3\pi}{2},\ 2\pi \).
