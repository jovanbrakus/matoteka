Izražavanje kosinusa preko trigonometrijskog izraza

Trigonometrijski identiteti i algebarska manipulacija

Tekst zadatka

Ako je \(\dfrac{1 - \sin x}{1 + \sin x} = a\) \(\left(\dfrac{\pi}{2} < x < \pi\right)\), tada je \(\cos x\) jednak:

(A) \(a - 1\)
(B) \(-\sqrt{1 - a^2}\)
(C) \(-\dfrac{2\sqrt{a}}{1 + a}\)
(D) \(-\dfrac{2a}{1 + a^2}\)
(E) \(-\sqrt{a}\)

Plan rešavanja

Iz date jednakosti izraziti \(\sin x\) preko \(a\), zatim koristeći osnovni trigonometrijski identitet \(\sin^2 x + \cos^2 x = 1\) izračunati \(\cos^2 x\), i na kraju odrediti predznak \(\cos x\) na osnovu intervala \(\left(\frac{\pi}{2}, \pi\right)\).

Podsetnik iz teorije

Ključne formule i teoreme
  • Pitagorin trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Znakovi trigonometrijskih funkcija: U drugom kvadrantu \(\left(\frac{\pi}{2} < x < \pi\right)\): \(\sin x > 0\) i \(\cos x < 0\)
  • Razlika kvadrata: \(A^2 - B^2 = (A-B)(A+B)\)
  • Opseg za \(a\): Pošto je \(0 < \sin x < 1\) u datom intervalu, imamo \(0 < 1 - \sin x < 1 + \sin x\), pa je \(0 < a < 1\).

Vizuelni prikaz

Grafik funkcije \(\cos x = -\frac{2\sqrt{a}}{1+a}\) u zavisnosti od parametra \(a \in (0, 1)\), sa oznakom znakova u drugom kvadrantu.

Rešenje korak po korak

1

Izražavanje \(\sin x\) preko \(a\)

Polazimo od date jednakosti i množimo obe strane sa \((1 + \sin x)\):

\[\frac{1 - \sin x}{1 + \sin x} = a\]
\[1 - \sin x = a(1 + \sin x) = a + a\sin x\]

Grupisanjem članova sa \(\sin x\) na jednu stranu:

\[1 - a = \sin x + a\sin x = \sin x(1 + a)\]

Odakle dobijamo:

\[\sin x = \frac{1 - a}{1 + a}\]
Proveri sebe: Da li je ova vrednost \(\sin x\) u intervalu \((0, 1)\)?

Pošto je \(0 < a < 1\), imamo \(0 < 1 - a < 1 + a\), pa je \(0 < \frac{1-a}{1+a} < 1\). Dakle, \(\sin x\) je pozitivan i manji od 1, što odgovara drugom kvadrantu.

2

Izračunavanje \(\cos^2 x\)

Koristimo Pitagorin identitet:

\[\cos^2 x = 1 - \sin^2 x = 1 - \left(\frac{1-a}{1+a}\right)^2\]

Svodimo na zajednički imenilac:

\[\cos^2 x = \frac{(1+a)^2 - (1-a)^2}{(1+a)^2}\]

Brojilac razvijamo koristeći razliku kvadrata \(A^2 - B^2 = (A-B)(A+B)\):

\[(1+a)^2 - (1-a)^2 = \bigl[(1+a) - (1-a)\bigr]\bigl[(1+a) + (1-a)\bigr] = 2a \cdot 2 = 4a\]

Dakle:

\[\cos^2 x = \frac{4a}{(1+a)^2}\]
3

Određivanje predznaka i izvlačenje korena

Pošto je \(x \in \left(\frac{\pi}{2}, \pi\right)\), ugao \(x\) je u drugom kvadrantu, gde je \(\cos x < 0\).

Izvlačimo koren i biramo negativnu vrednost:

\[\cos x = -\sqrt{\frac{4a}{(1+a)^2}} = -\frac{2\sqrt{a}}{1+a}\]

Napomena: Pošto je \(a > 0\) (tačnije \(0 < a < 1\)), koren \(\sqrt{a}\) je dobro definisan i pozitivan, a celokupan izraz je negativan, što je u skladu sa drugim kvadrantom.

Ključni uvid

Ključ za ovaj zadatak je pažljiva analiza predznaka. U drugom kvadrantu sinus je pozitivan, a kosinus negativan. Kada izračunamo \(\cos^2 x\), moramo pravilno izabrati negativni koren. Mnogi ponuđeni odgovori izgledaju slično, ali se razlikuju u predznaku ili u strukturi — samo odgovor (C) ima ispravnu kombinaciju: negativan predznak, koren iz \(a\), i pravilan imenilac \((1+a)\).

Konačan odgovor

\(\cos x = -\dfrac{2\sqrt{a}}{1 + a}\)

(A) \(a - 1\) (B) \(-\sqrt{1 - a^2}\) (C) \(-\dfrac{2\sqrt{a}}{1 + a}\) (D) \(-\dfrac{2a}{1 + a^2}\) (E) \(-\sqrt{a}\)

Verifikacija: Uzmimo \(x = \frac{2\pi}{3}\) (ugao od \(120°\)). Tada je \(\sin x = \frac{\sqrt{3}}{2}\), pa:

\(a = \frac{1 - \frac{\sqrt{3}}{2}}{1 + \frac{\sqrt{3}}{2}} = \frac{2 - \sqrt{3}}{2 + \sqrt{3}} = \frac{(2-\sqrt{3})^2}{(2+\sqrt{3})(2-\sqrt{3})} = \frac{7 - 4\sqrt{3}}{1} = 7 - 4\sqrt{3}\)

Očekivani rezultat: \(\cos\frac{2\pi}{3} = -\frac{1}{2}\). Proverom: \(-\frac{2\sqrt{7-4\sqrt{3}}}{1 + 7 - 4\sqrt{3}} = -\frac{2\sqrt{7-4\sqrt{3}}}{8 - 4\sqrt{3}}\).

Pošto je \(\sqrt{7-4\sqrt{3}} = 2 - \sqrt{3}\) (jer \((2-\sqrt{3})^2 = 7 - 4\sqrt{3}\)):

\(-\frac{2(2-\sqrt{3})}{4(2-\sqrt{3})} = -\frac{2}{4} = -\frac{1}{2}\). Tačno!

Česte greške

  • Pogrešan predznak: Zaboravljanje da je \(\cos x < 0\) u drugom kvadrantu. Bez analize predznaka, učenik može izabrati pozitivnu vrednost koja ne odgovara nijednom ponuđenom odgovoru.
  • Brkanje \((1+a)^2\) i \(1 + a^2\): Imenilac u \(\cos^2 x = \frac{4a}{(1+a)^2}\) sadrži \((1+a)^2\), a ne \(1 + a^2\). Ova razlika razdvaja tačan odgovor (C) od netačnog (D).
  • Greška kod razlike kvadrata: Razvijanje \((1+a)^2 - (1-a)^2\) mora dati \(4a\), ne \(4a^2\) ili nešto drugo. Koristite formulu \(A^2 - B^2 = (A-B)(A+B)\) za sigurnije računanje.
  • Ignorisanje uslova za \(a\): Mora se proveriti da je \(a > 0\) kako bi \(\sqrt{a}\) bio definisan. Ovo sledi iz uslova \(\frac{\pi}{2} < x < \pi\).

Dodatni izazov

Ako je \(\dfrac{1 - \cos x}{1 + \cos x} = b\) gde \(0 < x < \pi\), izrazi \(\sin x\) u funkciji od \(b\).

Pogledaj nagoveštaj

Postupak je analogan: izrazi \(\cos x\) preko \(b\), zatim koristi \(\sin^2 x = 1 - \cos^2 x\). Pazi na predznak — u prvom i drugom kvadrantu \(\sin x > 0\).
