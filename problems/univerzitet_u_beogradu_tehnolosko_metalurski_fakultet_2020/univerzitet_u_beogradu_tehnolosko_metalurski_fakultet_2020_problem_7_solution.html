Deljivost polinoma četvrtog stepena

Algebra — deljivost polinoma, identifikacija koeficijenata, faktorizacija

Tekst zadatka

Zbir brojeva \( a \) i \( b \) takvih da je polinom \( x^4 + ax^2 + b \) deljiv polinomom \( x^2 + 2x + 4 \) jednak je:

Polinom \(P(x)\) \(x^4 + ax^2 + b\)
Delilac \(D(x)\) \(x^2 + 2x + 4\)
Uslov \(D(x) \mid P(x)\)
Tražimo \(a + b\)
(A) \(19\)
(B) \(20\)
(C) \(17\)
(D) \(16\)
(E) \(12\)

Plan rešavanja

Pošto je \(P(x)\) stepena 4, a delilac \(D(x)\) stepena 2, količnik mora biti polinom stepena 2. Pretpostavimo da je količnik \(x^2 + cx + d\). Pomnožimo \((x^2 + 2x + 4)(x^2 + cx + d)\) i izjednačimo odgovarajuće koeficijente sa polaznim polinomom \(x^4 + ax^2 + b\). Iz uslova da koeficijenti uz \(x^3\) i \(x\) moraju biti nula, dobijamo \(c\) i \(d\), a zatim računamo \(a\), \(b\) i njihov zbir.

Podsetnik iz teorije

Ključne formule i pojmovi
  • Deljivost polinoma: Polinom \(P(x)\) je deljiv polinomom \(D(x)\) ako postoji polinom \(Q(x)\) takav da je \(P(x) = D(x) \cdot Q(x)\). Stepen količnika je \(\deg P - \deg D\).
  • Metod neodređenih koeficijenata: Za zadat oblik količnika sa nepoznatim koeficijentima \(c, d, \ldots\), izvršimo množenje i izjednačimo koeficijente uz iste stepene od \(x\) sa obe strane.
  • Sophie Germain identitet (specijalan slučaj): \( x^4 + 4y^4 = (x^2 + 2xy + 2y^2)(x^2 - 2xy + 2y^2) \). Za \(y = \sqrt{2}\) dobijamo \( x^4 + 16 = (x^2 + 2\sqrt{2}\,x + 4)(x^2 - 2\sqrt{2}\,x + 4) \), a opštije \( x^4 + 4x^2 + 16 = (x^2 + 2x + 4)(x^2 - 2x + 4) \).
  • Stepen polinoma kod proizvoda: \(\deg(P \cdot Q) = \deg P + \deg Q\).

Vizuelni prikaz

Šema množenja \((x^2 + 2x + 4)(x^2 + cx + d)\) i izjednačavanje koeficijenata sa \(x^4 + 0\cdot x^3 + a\cdot x^2 + 0\cdot x + b\).

Tabela proizvoda (Cauchy-jeva šema) i sistem jednačina za koeficijente

Rešenje korak po korak

1

Postavljanje količnika sa nepoznatim koeficijentima

Pošto je polinom \(P(x) = x^4 + ax^2 + b\) stepena \(4\), a delilac \(x^2 + 2x + 4\) stepena \(2\), količnik mora biti polinom stepena \(2\). Vodeći koeficijent količnika je \(1\) (jer su vodeći koeficijenti i polaznog polinoma i delioca jednaki \(1\)). Pretpostavimo:

\[ x^4 + ax^2 + b = (x^2 + 2x + 4)(x^2 + cx + d) \]

Cilj: odrediti \(c\) i \(d\), a zatim \(a\) i \(b\).

2

Razvijanje desne strane

Pomnožimo polinome član po član:

\[ \begin{aligned} (x^2 + 2x + 4)(x^2 + cx + d) &= x^2(x^2 + cx + d) + 2x(x^2 + cx + d) + 4(x^2 + cx + d) \\ &= x^4 + cx^3 + dx^2 + 2x^3 + 2cx^2 + 2dx + 4x^2 + 4cx + 4d \\ &= x^4 + (c+2)x^3 + (d + 2c + 4)x^2 + (2d + 4c)x + 4d. \end{aligned} \]

Grupisali smo članove uz iste stepene od \(x\).

3

Izjednačavanje koeficijenata

Sa leve strane imamo \(x^4 + 0\cdot x^3 + a\cdot x^2 + 0\cdot x + b\). Izjednačimo koeficijente uz iste stepene:

\[ \begin{cases} x^3: & c + 2 = 0 \\ x^2: & d + 2c + 4 = a \\ x^1: & 2d + 4c = 0 \\ x^0: & 4d = b \end{cases} \]
Proveri sebe: zašto su koeficijenti uz \(x^3\) i \(x\) jednaki nuli?

Polazni polinom je \(x^4 + ax^2 + b\) — on uopšte ne sadrži članove uz \(x^3\) ni uz \(x\). To znači da su njihovi koeficijenti tačno \(0\), pa to mora važiti i nakon razvijanja desne strane.

4

Određivanje \(c\) i \(d\)

Iz prve jednačine (\(x^3\)):

\[ c + 2 = 0 \;\Longrightarrow\; c = -2. \]

Iz treće jednačine (\(x^1\)) i upravo nađenog \(c = -2\):

\[ 2d + 4c = 0 \;\Longrightarrow\; 2d + 4 \cdot (-2) = 0 \;\Longrightarrow\; 2d = 8 \;\Longrightarrow\; d = 4. \]

Količnik je dakle \(x^2 - 2x + 4\).

5

Računanje \(a\) i \(b\)

Iz druge jednačine (\(x^2\)):

\[ a = d + 2c + 4 = 4 + 2 \cdot (-2) + 4 = 4 - 4 + 4 = 4. \]

Iz četvrte jednačine (\(x^0\)):

\[ b = 4d = 4 \cdot 4 = 16. \]
Proveri sebe: koliko iznosi \(a + b\)?

\(a + b = 4 + 16 = 20\). Tačan odgovor je \( (B)\ 20 \).

6

Provera množenjem

Proverimo da je zaista \( (x^2 + 2x + 4)(x^2 - 2x + 4) = x^4 + 4x^2 + 16 \):

\[ \begin{aligned} (x^2 + 2x + 4)(x^2 - 2x + 4) &= \big[(x^2 + 4) + 2x\big]\big[(x^2 + 4) - 2x\big] \\ &= (x^2 + 4)^2 - (2x)^2 \\ &= x^4 + 8x^2 + 16 - 4x^2 \\ &= x^4 + 4x^2 + 16. \end{aligned} \]

Iskoristili smo identitet \((u+v)(u-v) = u^2 - v^2\) sa \(u = x^2 + 4\) i \(v = 2x\). Dobijamo upravo \(x^4 + ax^2 + b\) za \(a = 4, b = 16\).

Ključni uvid

Kada je delilac kvadratni polinom oblika \(x^2 + px + q\), koristan trik je primetiti da je \(x^4 + (q - p^2 + 2q)\,x^2 + q^2 \) često moguće faktorisati kao \((x^2 + px + q)(x^2 - px + q)\). Za \(p = 2, q = 4\) dobijamo upravo \(x^4 + 4x^2 + 16\). Ovo je posebna pojava kod tzv. Sophie Germain identiteta.

\[ (x^2 + px + q)(x^2 - px + q) = (x^2 + q)^2 - (px)^2 = x^4 + (2q - p^2)x^2 + q^2. \]

Konačan odgovor

\[ a + b = 4 + 16 = 20 \]

(A) \(19\) (B) \(20\) (C) \(17\) (D) \(16\) (E) \(12\)

Verifikacija: za \(a = 4, b = 16\) imamo \(P(x) = x^4 + 4x^2 + 16 = (x^2 + 2x + 4)(x^2 - 2x + 4)\). Faktor \(x^2 + 2x + 4\) zaista deli \(P(x)\), pa je uslov ispunjen. Zbir je \(a + b = 20\).

Česte greške

  • Greška 1: Ignorisati uslov da koeficijenti uz \(x^3\) i \(x\) moraju biti nula. Bez ta dva uslova, sistem se ne može rešiti i zadatak ostaje neodređen.
  • Greška 2: Pretpostavka da je količnik oblika \(x^2 + d\) (bez člana sa \(x\)). To nije tačno: pošto delilac ima član \(2x\), neutralisanje koeficijenta uz \(x^3\) zahteva da količnik takođe ima član sa \(x\) (\(c \neq 0\)).
  • Greška 3: Pogrešno množenje: zaboraviti neki član u distribuciji ili pogrešiti znak. Preporučljivo je množenje izvoditi tabelarno (Cauchy-jeva šema na slici).
  • Greška 4: Pomešati \(a + b\) sa samim \(b\) i izabrati odgovor (D) \(16\).

Dodatni izazov

Odredi sve realne brojeve \(a\) i \(b\) takve da je polinom \(x^4 + ax^2 + b\) deljiv polinomom \(x^2 + 3x + 9\). Koliko je \(a + b\) u tom slučaju?

Pogledaj nagoveštaj

Postupi analogno: pretpostavi da je količnik \(x^2 + cx + d\). Iz koeficijenta uz \(x^3\) dobijaš \(c = -3\). Iz koeficijenta uz \(x\) dobijaš \(2d + 6c = 0\), tj. \(d = 9\). Tada je \(a = d + 2c + 9 = 9 - 6 + 9 = 12\) i \(b = 9d = 81\), pa je \(a + b = 93\). Provera: \((x^2 + 3x + 9)(x^2 - 3x + 9) = (x^2 + 9)^2 - 9x^2 = x^4 + 9x^2 + 81\). Tačno.
