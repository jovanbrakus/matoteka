Opadajuća aritmetička progresija sa zadatim uslovima

Nizovi — aritmetička progresija, sistem jednačina, uslov opadanja

📋 Tekst zadatka

Opadajuća aritmetička progresija \((a_n)\) je takva da važi \(a_1^2 + a_2^2 + a_3^2 = 56\) i \(\dfrac{a_{10}}{a_2} = 5\). Tada je \(a_{2014}\) jednako:

(A) \(-4028\)
(B) \(4028\)
(C) \(4030\)
(D) \(-4030\)
(E) takva progresija ne postoji

🎯 Plan rešavanja

Označimo prvi član sa \(a\) i razliku sa \(d\). Iz uslova \(\frac{a_{10}}{a_2} = 5\) izražavamo vezu između \(a\) i \(d\). Zatim iz uslova \(a_1^2 + a_2^2 + a_3^2 = 56\) nalazimo konkretne vrednosti. Uslov opadanja (\(d < 0\)) eliminiše jedno rešenje. Na kraju računamo \(a_{2014}\).

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Opšti član aritmetičke progresije: \(a_n = a_1 + (n-1)d\), gde je \(d\) zajednička razlika.
  • Opadajuća progresija: Progresija je opadajuća ako i samo ako je \(d < 0\).
  • Rastuća progresija: Progresija je rastuća ako i samo ako je \(d > 0\).
  • Veza uzastopnih članova: \(a_2 = a_1 + d\), \(a_3 = a_1 + 2d\), \(a_{10} = a_1 + 9d\).

📐 Vizuelni prikaz aritmetičke progresije

Grafik opadajuće aritmetičke progresije \(a_n = -2 + (n-1) \cdot (-2) = -2n\). Prikazani su ključni članovi: \(a_1, a_2, a_3, a_{10}\), i put do \(a_{2014} = -4028\).

✏️ Rešenje korak po korak

1

Postavljamo oznake

Neka je \(a_1 = a\) i razlika progresije \(d\). Tada:

\[ a_2 = a + d, \quad a_3 = a + 2d, \quad a_{10} = a + 9d \]
2

Koristimo uslov \(\frac{a_{10}}{a_2} = 5\)

Uvrštavamo izraze za \(a_{10}\) i \(a_2\):

\[ \frac{a + 9d}{a + d} = 5 \]

Množimo sa \((a + d)\):

\[ a + 9d = 5(a + d) = 5a + 5d \]
\[ a + 9d - 5a - 5d = 0 \implies -4a + 4d = 0 \implies d = a \]
Proveri sebe: Šta znači \(d = a\)?

Razlika progresije je jednaka prvom članu. To znači da su članovi: \(a_1 = a\), \(a_2 = 2a\), \(a_3 = 3a\), i uopšte \(a_n = na\). Progresija je potpuno određena jednim parametrom!

3

Koristimo uslov \(a_1^2 + a_2^2 + a_3^2 = 56\)

Zamenom \(d = a\) dobijamo \(a_1 = a\), \(a_2 = 2a\), \(a_3 = 3a\):

\[ a^2 + (2a)^2 + (3a)^2 = 56 \]
\[ a^2 + 4a^2 + 9a^2 = 56 \]
\[ 14a^2 = 56 \implies a^2 = 4 \implies a = \pm 2 \]
4

Primenjujemo uslov opadanja

Progresija je opadajuća, što znači \(d < 0\). Kako je \(d = a\):

  • Ako \(a = 2\), onda \(d = 2 > 0\) — progresija raste. ✗
  • Ako \(a = -2\), onda \(d = -2 < 0\) — progresija opada. ✓

Dakle, \(a_1 = -2\) i \(d = -2\).

Proveri sebe: Da li važi \(\frac{a_{10}}{a_2} = 5\)?

\(a_{10} = -2 + 9 \cdot (-2) = -2 - 18 = -20\)

\(a_2 = -2 + (-2) = -4\)

\(\frac{a_{10}}{a_2} = \frac{-20}{-4} = 5\) ✓

5

Izračunavamo \(a_{2014}\)

\[ a_{2014} = a_1 + 2013 \cdot d = -2 + 2013 \cdot (-2) \]
\[ a_{2014} = -2 - 4026 = -4028 \]

Napomena: Može se koristiti i formula \(a_n = na\), jer je \(d = a = -2\):

\[ a_{2014} = 2014 \cdot (-2) = -4028 \]

💡 Ključni uvid

Uslov \(\frac{a_{10}}{a_2} = 5\) direktno daje vezu \(d = a\), čime se cela progresija svodi na \(a_n = na\). To drastično pojednostavljuje problem — umesto sistema sa dve nepoznate, rešavamo jednu jednačinu \(14a^2 = 56\). Zatim uslov opadanja bira jedino rešenje \(a = -2\).

✅ Konačan odgovor

\[ a_{2014} = -4028 \]

(A) \(-4028\) (B) \(4028\) (C) \(4030\) (D) \(-4030\) (E) takva progresija ne postoji

Provera: Za \(a_1 = -2, d = -2\): \(a_1^2 + a_2^2 + a_3^2 = 4 + 16 + 36 = 56\) ✓, \(\frac{a_{10}}{a_2} = \frac{-20}{-4} = 5\) ✓, \(d = -2 < 0\) (opadajuća) ✓.

⚠️ Česte greške

  • Zaboravljanje uslova opadanja: Bez ovog uslova, \(a = 2, d = 2\) daje \(a_{2014} = 4028\) (odgovor B). Uslov opadanja je ključan za izbor pravog rešenja.
  • Greška za jedan u formuli: \(a_{2014} = a_1 + \mathbf{2013} \cdot d\), ne \(2014 \cdot d\). Srećom, ovde je \(a_n = na\) pa oba pristupa daju isti rezultat, ali u opštem slučaju bi greška bila fatalna.
  • Pogrešna primena uslova količnika: Treba pažljivo rešiti \(\frac{a+9d}{a+d} = 5\). Česta greška je pogrešno prebacivanje članova sa jedne strane na drugu.
  • Odgovor \(-4030\): Pojavljuje se ako se pogrešno izračuna \(-2 - 2 \cdot 2014 = -4030\) umesto korektnog \(-2 \cdot 2014 = -4028\).

🏆 Dodatni izazov

Za istu progresiju (\(a_1 = -2, d = -2\)), odredite sumu prvih 100 članova \(S_{100}\). Koliko iznosi \(|S_{100}|\)?

Pogledaj nagoveštaj

Koristimo formulu za sumu: \(S_n = \frac{n}{2}(2a_1 + (n-1)d)\). Za \(n = 100\): \(S_{100} = \frac{100}{2}(2 \cdot (-2) + 99 \cdot (-2)) = 50 \cdot (-4 - 198) = 50 \cdot (-202) = -10100\). Dakle, \(|S_{100}| = 10100\).
