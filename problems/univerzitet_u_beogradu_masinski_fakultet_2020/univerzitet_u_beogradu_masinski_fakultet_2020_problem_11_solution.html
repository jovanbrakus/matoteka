Identitet sa zbirom kubova sinusa i kosinusa

Trigonometrija — faktorisanje, dvostruki ugao i skraćivanje

📋 Tekst zadatka

Izraz

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} \]

je identički jednak:

(A) \( \dfrac{\cos x - \sin x}{2} \)
(B) \( 1 \)
(C) \( \dfrac{\cos x}{2} \)
(D) \( \dfrac{1 + \cos^2 x}{4} \)
(E) \( \dfrac{\sin x + \cos x}{2} \)

🎯 Plan rešavanja

Brojilac \( \sin^3 x + \cos^3 x \) je zbir kubova — faktorišemo ga formulom \( a^3 + b^3 = (a + b)(a^2 - ab + b^2) \), a zatim iskoristimo \( \sin^2 x + \cos^2 x = 1 \). Imenilac sredimo identitetom \( \sin 2x = 2\sin x \cos x \) — izbiće zajednički faktor. Posle skraćivanja ostaje čist izraz koji odgovara jednoj od ponuđenih opcija.

📚 Podsetnik iz teorije

Faktorisanje, osnovni i dvostruki uglovi
  • Zbir kubova:
    \[ a^3 + b^3 = (a + b)(a^2 - ab + b^2). \]
    Drugi faktor je „nepotpun kvadrat”, sa minusom ispred \( ab \).
  • Pitagorin trigonometrijski identitet:
    \[ \sin^2 x + \cos^2 x = 1 \quad (\forall x \in \mathbb{R}). \]
  • Sinus dvostrukog ugla:
    \[ \sin 2x = 2\sin x \cos x. \]
    Otud \( \sin x \cos x = \dfrac{\sin 2x}{2} \).
  • Ograničenje za \( \sin x \cos x \): kako \( |\sin 2x| \le 1 \), važi \( |\sin x \cos x| \le \dfrac{1}{2} \), pa je \( 1 - \sin x \cos x \ge \dfrac{1}{2} > 0 \) — ovaj izraz nikada nije nula i možemo njime skratiti.

📐 Vizuelni prikaz — grafovi se poklapaju

Plavom linijom je nacrtan polazni izraz \( y = \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} \), a zelenom isprekidanom linijom \( y = \dfrac{\sin x + \cos x}{2} \). Krive se savršeno poklapaju za svaki \( x \in [-2\pi, 2\pi] \) — to je grafička potvrda da je izraz identički jednak (E).

✏️ Rešenje korak po korak

1

Faktoriši brojilac kao zbir kubova

Primenjujemo formulu \( a^3 + b^3 = (a + b)(a^2 - ab + b^2) \), pri čemu je \( a = \sin x \), \( b = \cos x \):

\[ \sin^3 x + \cos^3 x = (\sin x + \cos x)\bigl(\sin^2 x - \sin x \cos x + \cos^2 x\bigr). \]
Proveri sebe: zašto u drugoj zagradi piše „minus \( ab \)”, a ne „plus”?

To je struktura formule za zbir kubova. Razlika kubova bi imala plus: \( a^3 - b^3 = (a - b)(a^2 + ab + b^2) \). Predznaci u zagradi suprotni su predznaku između \( a \) i \( b \) u prvom faktoru.

2

Sredi drugu zagradu pomoću \( \sin^2 + \cos^2 = 1 \)

Pošto je \( \sin^2 x + \cos^2 x = 1 \), druga zagrada postaje:

\[ \sin^2 x - \sin x \cos x + \cos^2 x = 1 - \sin x \cos x. \]

Otud kompletan brojilac glasi:

\[ \sin^3 x + \cos^3 x = (\sin x + \cos x)\bigl(1 - \sin x \cos x\bigr). \]
3

Sredi imenilac iskoristivši \( \sin 2x = 2\sin x \cos x \)

Pošto je \( \sin 2x = 2\sin x \cos x \), zamenjujemo i izvlačimo zajednički faktor:

\[ 2 - \sin 2x = 2 - 2\sin x \cos x = 2\bigl(1 - \sin x \cos x\bigr). \]
Proveri sebe: koju zagradu vidiš i u brojiocu i u imeniocu?

Faktor \( 1 - \sin x \cos x \) — pojavljuje se i gore i dole, što znači da se može skratiti. Ovo je „klizav” detalj koji rešava ceo zadatak.

4

Skrati zajednički faktor

Ubacujemo dobijene oblike u razlomak:

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{(\sin x + \cos x)(1 - \sin x \cos x)}{2(1 - \sin x \cos x)}. \]

Faktor \( (1 - \sin x \cos x) \) skraćujemo (videti opravdanje u sledećem koraku). Ostaje:

\[ \dfrac{\sin x + \cos x}{2}. \]
5

Opravdanje skraćivanja: \( 1 - \sin x \cos x \neq 0 \)

Da bi skraćivanje važilo za svako \( x \), faktor \( 1 - \sin x \cos x \) ne sme biti nula. Ali:

\[ \sin x \cos x = \dfrac{\sin 2x}{2}, \quad |\sin 2x| \le 1 \;\Longrightarrow\; |\sin x \cos x| \le \dfrac{1}{2}. \]

Zato je \( 1 - \sin x \cos x \ge 1 - \dfrac{1}{2} = \dfrac{1}{2} > 0 \). Faktor je uvek pozitivan, nikada nula — skraćivanje je dozvoljeno za svako \( x \in \mathbb{R} \). Identitet važi globalno.

6

Numerička provera za dve vrednosti \( x \)

Brzo proverimo identitet na \( x = 0 \) i \( x = \dfrac{\pi}{4} \):

\[ \begin{aligned} x = 0:\quad &\dfrac{0 + 1}{2 - 0} = \dfrac{1}{2}, \quad \dfrac{0 + 1}{2} = \dfrac{1}{2}. \;\checkmark \\[4pt] x = \dfrac{\pi}{4}:\quad &\dfrac{2 \cdot (\sqrt{2}/2)^3}{2 - 1} = 2 \cdot \dfrac{\sqrt{2}}{4} = \dfrac{\sqrt{2}}{2},\quad \dfrac{\sqrt{2}/2 + \sqrt{2}/2}{2} = \dfrac{\sqrt{2}}{2}. \;\checkmark \end{aligned} \]

Obe strane se poklapaju — potvrda algebarskog rešenja.

💡 Ključni uvid

Trigonometrijski izraz je samo algebarski izraz u kome su slova \( \sin x \) i \( \cos x \) — pa „algebarski refleksi” poput zbira kubova rade kao i obično. Ključna ideja je uniformisati brojilac i imenilac pomoću dva osnovna identiteta (\( \sin^2 + \cos^2 = 1 \) i \( \sin 2x = 2\sin x \cos x \)) tako da se pojavi zajednički faktor koji se može skratiti.

⚠️ Česte greške

  • Pogrešno faktorisanje zbira kubova: pisanje \( a^3 + b^3 = (a + b)(a^2 + ab + b^2) \) (sa „plus \( ab \)” umesto „minus \( ab \)”). Mali znak — velika greška.
  • Skraćivanje bez provere da li je faktor različit od nule. U ovom zadatku faktor \( 1 - \sin x \cos x \) je uvek pozitivan, ali to treba pokazati — uvek se pozvati na ograničenje \( |\sin 2x| \le 1 \).
  • Nedovoljna provera odgovora: jedna numerička proba nije dovoljna! Za \( x = 0 \) opcije (A), (C), (D) i (E) sve daju \( 1/2 \). Tek za \( x = \pi/2 \) razlika je vidljiva (samo (E) ostaje).
  • Mešanje \( \sin 2x \) sa \( 2\sin x \) ili \( \sin^2 x \). Identitet je \( \sin 2x = 2\sin x \cos x \).
  • Pokušaj da se \( \sin^3 x + \cos^3 x \) zameni sa \( (\sin x + \cos x)^3 \) — pogrešno, jer \( (a+b)^3 = a^3 + 3a^2 b + 3ab^2 + b^3 \neq a^3 + b^3 \).

✅ Konačan odgovor

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{\sin x + \cos x}{2}. \]

(A) \( \dfrac{\cos x - \sin x}{2} \) (B) \( 1 \) (C) \( \dfrac{\cos x}{2} \) (D) \( \dfrac{1 + \cos^2 x}{4} \) (E) \( \dfrac{\sin x + \cos x}{2} \)

Provera za \( x = \dfrac{\pi}{2} \): leva strana \( = \dfrac{1 + 0}{2 - 0} = \dfrac{1}{2} \); desna strana \( = \dfrac{1 + 0}{2} = \dfrac{1}{2} \). \( \checkmark \) Opcije (A), (C), (D) za \( x = \dfrac{\pi}{2} \) daju \( -\dfrac{1}{2} \), \( 0 \), \( \dfrac{1}{4} \) — sve različito od \( \dfrac{1}{2} \). Tačan odgovor je (E).

🏆 Dodatni izazov

Pokaži da i sledeći izraz ima jednostavan zatvoreni oblik:

\[ \dfrac{\sin^3 x - \cos^3 x}{1 - \sin x \cos x}. \]

Pogledaj nagoveštaj

Korak 1. Razlika kubova: \( a^3 - b^3 = (a - b)(a^2 + ab + b^2) \). Sa \( a = \sin x \), \( b = \cos x \): brojilac \( = (\sin x - \cos x)(\sin^2 x + \sin x \cos x + \cos^2 x) = (\sin x - \cos x)(1 + \sin x \cos x) \).

Korak 2. Imenilac \( 1 - \sin x \cos x \) se ne poklapa sa \( 1 + \sin x \cos x \) — različiti su predznaci. Dakle nije sve isto kao u glavnom zadatku!

Korak 3. Razlomak postaje \( \dfrac{(\sin x - \cos x)(1 + \sin x \cos x)}{1 - \sin x \cos x} \). Ne skraćuje se direktno; možeš dalje algebarski srediti, ali se ne dobija tako kratak izraz kao u originalnom zadatku.

Pouka: uspeh u glavnom zadatku zavisi od „srećne” podudarnosti znaka — razlika kvadrata u zagradi (\( -\sin x \cos x \)) tačno odgovara faktoru \( 1 - \sin x \cos x \) u imeniocu.
