Pojednostavljenje trigonometrijskog izraza \( \dfrac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \)

Trigonometrija — zbir kubova, osnovni identitet \( \sin^{2} x + \cos^{2} x = 1 \), dvostruki ugao \( \sin 2x \).

📋 Tekst zadatka

Izraz

\[ \frac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \]

identički je jednak:

(A) \( \dfrac{\cos x - \sin x}{2} \)
(B) \( 1 \)
(C) \( \dfrac{\cos x}{2} \)
(D) \( \dfrac{1 + \cos^{2} x}{4} \)
(E) \( \dfrac{\sin x + \cos x}{2} \)

🎯 Plan rešavanja

Brojilac rastavimo formulom za zbir kubova, koristeći \( \sin^{2} x + \cos^{2} x = 1 \). Imenilac pretvorimo u proizvod preko \( \sin 2x = 2 \sin x \cos x \). Tada se isti činilac \( (1 - \sin x \cos x) \) pojavljuje u oba mesta — skratimo ga (uz proveru da nije nula) i ostaje čista jednostavna forma.

📚 Podsetnik iz teorije

Ključne formule koje koristimo
  • Zbir kubova \( a^{3} + b^{3} = (a + b)\,(a^{2} - ab + b^{2}) \). Sredinski član je sa minusom!
  • Pitagorin identitet \( \sin^{2} x + \cos^{2} x = 1 \) za svako realno \( x \).
  • Dvostruki ugao \( \sin 2x = 2 \sin x \cos x \).
  • Posledica: \( \sin x \cos x = \dfrac{1}{2}\sin 2x \in \Bigl[-\tfrac{1}{2},\; \tfrac{1}{2}\Bigr] \), pa \( 1 - \sin x \cos x \in \Bigl[\tfrac{1}{2},\; \tfrac{3}{2}\Bigr] > 0 \). Ovo je strogo pozitivno — opravdava skraćivanje na celom \( \mathbb{R} \).
  • Napomena: razlika kubova ima drugačiji znak — \( a^{3} - b^{3} = (a - b)\,(a^{2} + ab + b^{2}) \) (sredina sa plusom).

📐 Vizuelni prikaz

Grafik leve strane \( f(x) = \dfrac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \) (plava) i desne strane \( g(x) = \dfrac{\sin x + \cos x}{2} \) (zelena isprekidana) na intervalu \( [-\pi, \pi] \). Krive se poklapaju u svakoj tački — to je vizuelni dokaz identičnosti. Maksimalna vrednost \( \tfrac{\sqrt{2}}{2} \approx 0{,}707 \) dostiže se za \( x = \pi/4 \), a minimum \( -\tfrac{\sqrt{2}}{2} \) za \( x = -3\pi/4 \).

✏️ Rešenje korak po korak

1

Brojilac: razloži zbir kubova

Primenimo formulu \( a^{3} + b^{3} = (a+b)(a^{2} - ab + b^{2}) \) sa \( a = \sin x \), \( b = \cos x \):

\[ \sin^{3} x + \cos^{3} x = (\sin x + \cos x)\bigl(\sin^{2} x - \sin x \cos x + \cos^{2} x\bigr). \]

Sada upotrebimo Pitagorin identitet \( \sin^{2} x + \cos^{2} x = 1 \) — dva člana se „spajaju” u jedinicu:

\[ \sin^{3} x + \cos^{3} x = (\sin x + \cos x)\,(1 - \sin x \cos x). \]
Proveri sebe: zašto je sredinski član sa minusom?

Iz identiteta \( (a+b)(a^{2} - ab + b^{2}) = a^{3} + a^{2}b - a^{2}b - ab^{2} + ab^{2} + b^{3} = a^{3} + b^{3} \) — svi srednji članovi se međusobno potiru samo ako je sredinski koeficijent \( -ab \). Kod razlike kubova \( a^{3} - b^{3} \) sredina je sa plusom: \( (a-b)(a^{2} + ab + b^{2}) \). Lako se pomeša!

2

Imenilac: izvuci zajednički činilac iz \( 2 - \sin 2x \)

Formulom za dvostruki ugao \( \sin 2x = 2 \sin x \cos x \):

\[ 2 - \sin 2x = 2 - 2 \sin x \cos x = 2\,(1 - \sin x \cos x). \]

Pojavio se isti činilac \( (1 - \sin x \cos x) \) kao u brojiocu — to je signal da nas čeka lepo skraćivanje.

Proveri sebe: da li je \( \sin 2x = \sin x \cos x \) (bez dvojke)?

Ne. Tačna formula glasi \( \sin 2x = 2 \sin x \cos x \). Ovo je čest izvor grešaka — uvek ima faktor \( 2 \). Lako se zapamti iz \( \sin(A+B) = \sin A \cos B + \cos A \sin B \) za \( A = B = x \).

3

Skrati zajednički činilac

Spojimo Korake 1 i 2:

\[ \frac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} = \frac{(\sin x + \cos x)\,(1 - \sin x \cos x)}{2\,(1 - \sin x \cos x)}. \]

Pre skraćivanja moramo da znamo da \( (1 - \sin x \cos x) \neq 0 \). Pošto je \( \sin x \cos x = \tfrac{1}{2}\sin 2x \in \bigl[-\tfrac{1}{2},\, \tfrac{1}{2}\bigr] \), sledi

\[ 1 - \sin x \cos x \in \Bigl[\tfrac{1}{2},\; \tfrac{3}{2}\Bigr] \;>\; 0 \quad \text{za svako } x \in \mathbb{R}. \]

Skraćivanje je dakle uvek dozvoljeno:

\[ \frac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} = \frac{\sin x + \cos x}{2}. \]
4

Numerička provera (\( x = \pi/4 \))

Za \( x = \pi/4 \) je \( \sin x = \cos x = \tfrac{\sqrt{2}}{2} \) i \( \sin 2x = \sin\tfrac{\pi}{2} = 1 \). Polazni izraz:

\[ \frac{2 \cdot \bigl(\tfrac{\sqrt{2}}{2}\bigr)^{3}}{2 - 1} = \frac{2 \cdot \tfrac{2\sqrt{2}}{8}}{1} = \frac{\sqrt{2}}{2}. \]

Naš rezultat \( \dfrac{\sin x + \cos x}{2} = \dfrac{\sqrt{2}/2 + \sqrt{2}/2}{2} = \dfrac{\sqrt{2}}{2} \). Slaže se.

Druga provera, \( x = 0 \): polazni izraz \( = \dfrac{0 + 1}{2 - 0} = \dfrac{1}{2} \); rezultat \( = \dfrac{0 + 1}{2} = \dfrac{1}{2} \). Slaže se.

💡 Ključni uvid

Kada vidiš zbir kubova trigonometrijskih funkcija, prvo razloži formulom \( a^{3} + b^{3} = (a+b)(a^{2} - ab + b^{2}) \). Pošto je \( \sin^{2} + \cos^{2} = 1 \), kvadratni faktor se uvek pojednostavljuje u \( 1 - \sin x \cos x \). U paru sa imeniocem oblika \( 2 - \sin 2x = 2(1 - \sin x \cos x) \), isti činilac se pojavljuje na oba mesta — to je „skriveni manevar” koji projektanti zadatka koriste da bi izraz delovao komplikovano dok zapravo skraćivanje sve čisti.

✅ Konačan odgovor

\[ \frac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \;\equiv\; \frac{\sin x + \cos x}{2} \]

(A) \( \dfrac{\cos x - \sin x}{2} \) (B) \( 1 \) (C) \( \dfrac{\cos x}{2} \) (D) \( \dfrac{1 + \cos^{2} x}{4} \) (E) \( \dfrac{\sin x + \cos x}{2} \)

Provera za tri vrednosti — \( x = 0 \): \( \tfrac{0 + 1}{2 - 0} = \tfrac{1}{2} = \tfrac{\sin 0 + \cos 0}{2} \). \( x = \pi/2 \): \( \tfrac{1 + 0}{2 - 0} = \tfrac{1}{2} = \tfrac{1 + 0}{2} \). \( x = \pi/4 \): \( \tfrac{\sqrt{2}/2}{1} = \tfrac{\sqrt{2}}{2} = \tfrac{\sqrt{2}/2 + \sqrt{2}/2}{2} \). Sve tri vrednosti su konzistentne sa opcijom (E); ostale opcije ne prolaze ni za \( x = \pi/4 \).

⚠️ Česte greške

  • Pogrešna formula za zbir kubova. \( a^{3} + b^{3} \neq (a+b)^{3} \). Tačno: \( (a+b)(a^{2} - ab + b^{2}) \) — sredina je sa minusom. Kod razlike kubova \( a^{3} - b^{3} = (a-b)(a^{2} + ab + b^{2}) \) sredina je sa plusom; brkanje znakova je čest izvor greške.
  • Izostavljen faktor 2 u \( \sin 2x \). Pisanje \( \sin 2x = \sin x \cos x \) (bez dvojke) razbija ceo zadatak. Tačno: \( \sin 2x = 2 \sin x \cos x \).
  • Skraćivanje bez provere da činilac nije nula. Algebarsko skraćivanje \( \tfrac{A \cdot B}{C \cdot B} = \tfrac{A}{C} \) važi samo kada \( B \neq 0 \). Ovde, srećom, \( 1 - \sin x \cos x > 0 \) za svako \( x \), pa je identitet definisan na celom \( \mathbb{R} \) — ali to treba proveriti, ne pretpostaviti.
  • „Sviranje sa znakovima” u brojiocu. Klopka (A) \( \tfrac{\cos x - \sin x}{2} \) razlikuje se od tačnog odgovora samo po znaku jednog sabirka. Za \( x = \pi/4 \) opcija (A) daje \( 0 \), dok tačan izraz daje \( \tfrac{\sqrt{2}}{2} \) — tako se klopka odmah „otkriva” jednom testnom vrednošću.
  • Pokušaj „brutalne sile” preko razvoja sin³ i cos³ u sin 3x / cos 3x. Postoje formule \( \sin^{3} x = \tfrac{3 \sin x - \sin 3x}{4} \) i \( \cos^{3} x = \tfrac{3 \cos x + \cos 3x}{4} \), ali one komplikuju stvar — direktno faktorisanje zbira kubova je elegantnije i brže.

Pogledaj nagoveštaj

Iskoristi \( a^{3} - b^{3} = (a - b)(a^{2} + ab + b^{2}) \): brojilac je \( (\sin x - \cos x)(1 + \sin x \cos x) \). Imenilac \( 2 + \sin 2x = 2(1 + \sin x \cos x) \). Pošto je \( 1 + \sin x \cos x \in [1/2,\, 3/2] > 0 \), skraćivanje opet važi svuda, i izraz se sažima u \( \dfrac{\sin x - \cos x}{2} \). Strukturno je isti trik: faktorisanje kubova + Pitagorin identitet + dvostruki ugao.
