Vrednost izraza \( \sin^4 x + \cos^4 x \)

Trigonometrija — osnovni identitet \( \sin^2 + \cos^2 = 1 \), simetrične sume parnih stepena

📋 Tekst zadatka

Ako je \( \operatorname{tg} x = 2 \), onda je vrednost izraza

\[ \sin^4 x + \cos^4 x \]

jednaka:

(A) \( \dfrac{4}{25} \)
(B) \( \dfrac{7}{25} \)
(C) \( \dfrac{4}{5} \)
(D) \( \dfrac{17}{25} \)
(E) \( 1 \)

🎯 Plan rešavanja

Iz \( \operatorname{tg} x = 2 \) izrazimo \( \sin x = 2 \cos x \) i, koristeći \( \sin^2 x + \cos^2 x = 1 \), nađemo \( \cos^2 x = \tfrac{1}{5} \) i \( \sin^2 x = \tfrac{4}{5} \). Zatim direktno izračunamo \( \sin^4 x + \cos^4 x \) kao zbir kvadrata, ili upotrebimo identitet \( \sin^4 x + \cos^4 x = 1 - 2\sin^2 x \cos^2 x \).

📚 Podsetnik iz teorije

Ključne formule
  • Pitagorin identitet: \( \sin^2 x + \cos^2 x = 1 \) — temelj svih ovakvih zadataka.
  • Definicija tangensa: \( \operatorname{tg} x = \dfrac{\sin x}{\cos x} \) (za \( \cos x \ne 0 \)).
  • Identitet kvadrata zbira: \( (a^2 + b^2)^2 = a^4 + 2a^2 b^2 + b^4 \), pa
  • \( \sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x = 1 - 2\sin^2 x \cos^2 x. \)
  • Korisno: \( 1 + \operatorname{tg}^2 x = \dfrac{1}{\cos^2 x} \), pa direktno \( \cos^2 x = \dfrac{1}{1 + \operatorname{tg}^2 x} \).

📐 Vizuelni prikaz

Pravougli trougao sa katetama \( 1 \) (nalegla, uz ugao \( x \)) i \( 2 \) (naspramna). Tada je \( \operatorname{tg} x = \tfrac{2}{1} = 2 \), hipotenuza je \( \sqrt{5} \), pa je \( \sin x = \tfrac{2}{\sqrt{5}} \) i \( \cos x = \tfrac{1}{\sqrt{5}} \). Iz toga \( \sin^2 x = \tfrac{4}{5} \), \( \cos^2 x = \tfrac{1}{5} \), i traženi zbir je \( \tfrac{16}{25} + \tfrac{1}{25} = \tfrac{17}{25} \).

✏️ Rešenje korak po korak

1

Iz \( \operatorname{tg} x = 2 \) do \( \sin x = 2\cos x \)

Po definiciji tangensa važi \( \operatorname{tg} x = \dfrac{\sin x}{\cos x} = 2 \), pa je \( \cos x \ne 0 \) i

\[ \sin x = 2 \cos x. \]
Proveri sebe: zašto \( \cos x \ne 0 \)?

Tangens je definisan samo kada je \( \cos x \ne 0 \). Pošto je dato \( \operatorname{tg} x = 2 \), to je automatski ispunjeno.

2

Pitagorin identitet daje \( \cos^2 x \)

Zamenimo \( \sin x = 2 \cos x \) u osnovni identitet:

\[ \sin^2 x + \cos^2 x = 1 \;\Longrightarrow\; (2\cos x)^2 + \cos^2 x = 1, \]
\[ 4\cos^2 x + \cos^2 x = 5\cos^2 x = 1 \;\Longrightarrow\; \cos^2 x = \frac{1}{5}. \]

A onda

\[ \sin^2 x = 1 - \cos^2 x = 1 - \tfrac{1}{5} = \tfrac{4}{5}. \]

Alternativno, brzom formulom \( \cos^2 x = \dfrac{1}{1+\operatorname{tg}^2 x} = \dfrac{1}{1+4} = \tfrac{1}{5} \).

3

Direktan račun \( \sin^4 x + \cos^4 x \)

Pošto su \( \sin^2 x \) i \( \cos^2 x \) sada poznati brojevi, kvadriramo svaki:

\[ \sin^4 x = (\sin^2 x)^2 = \left(\tfrac{4}{5}\right)^2 = \tfrac{16}{25}, \qquad \cos^4 x = (\cos^2 x)^2 = \left(\tfrac{1}{5}\right)^2 = \tfrac{1}{25}. \]
\[ \sin^4 x + \cos^4 x = \tfrac{16}{25} + \tfrac{1}{25} = \tfrac{17}{25}. \]
4

Provera preko identiteta

Korisno je proveriti rezultat drugim putem:

\[ \sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2 \sin^2 x \cos^2 x = 1 - 2 \cdot \tfrac{4}{5} \cdot \tfrac{1}{5} = 1 - \tfrac{8}{25} = \tfrac{17}{25}. \;\checkmark \]
Proveri sebe: koliko je \( \sin(2x) \) ovde?

\( \sin 2x = 2 \sin x \cos x \), pa \( \sin^2(2x) = 4 \sin^2 x \cos^2 x = 4 \cdot \tfrac{4}{25} = \tfrac{16}{25} \). Tada \( \sin^4 x + \cos^4 x = 1 - \tfrac{1}{2}\sin^2(2x) = 1 - \tfrac{8}{25} = \tfrac{17}{25}. \)

💡 Ključni uvid

Trik je u tome da iz date vrednosti \( \operatorname{tg} x \) ne moramo da nađemo sam ugao \( x \) — dovoljno nam je \( \sin^2 x \) i \( \cos^2 x \). Za parne stepene znak ne igra ulogu, pa odgovor zavisi samo od \( \operatorname{tg}^2 x \). Identitet \( \sin^4 + \cos^4 = 1 - 2\sin^2 \cos^2 \) skraćuje račun jer odmah povezuje sve sa Pitagorinim identitetom.

✅ Konačan odgovor

\[ \sin^4 x + \cos^4 x = \tfrac{16}{25} + \tfrac{1}{25} = \tfrac{17}{25} \]

(A) \( \dfrac{4}{25} \) (B) \( \dfrac{7}{25} \) (C) \( \dfrac{4}{5} \) (D) \( \dfrac{17}{25} \) (E) \( 1 \)

Provera: \( 1 - 2\sin^2 x \cos^2 x = 1 - 2 \cdot \tfrac{4}{5} \cdot \tfrac{1}{5} = 1 - \tfrac{8}{25} = \tfrac{17}{25} \). ✓

⚠️ Česte greške

  • Pogrešno kvadriranje razlomka: \( \left(\tfrac{4}{5}\right)^2 = \tfrac{16}{25} \), ne \( \tfrac{8}{25} \) niti \( \tfrac{4}{25} \).
  • Zaboravljen sredinski član: \( (\sin^2 + \cos^2)^2 \ne \sin^4 + \cos^4 \) — postoji još \( 2 \sin^2 \cos^2 \).
  • Mešanje stepena: \( \sin^4 x \ne (\sin x)^4 \) je tačno isto, ali nekad se piše \( \sin x^4 = \sin(x^4) \) — ovo je greška u zapisu.
  • Pretpostavka znaka: ne treba pretpostavljati da je \( \sin x > 0 \) — ovde traženi izraz zavisi samo od \( \sin^2 x \) i \( \cos^2 x \), pa rezultat ne zavisi od kvadranta.
  • Greška u formuli \( 1 + \operatorname{tg}^2 = 1/\cos^2 \): ponekad se napiše \( 1 + \operatorname{tg}^2 = \cos^2 \) — pogrešno; treba \( = 1/\cos^2 \).

🏆 Dodatni izazov

Ako je \( \operatorname{tg} x = 2 \), izračunaj vrednost izraza \( \sin^6 x + \cos^6 x \). Možeš li opštu formulu napisati samo preko \( \operatorname{tg} x \)?

Pogledaj nagoveštaj

Identitet: \( \sin^6 + \cos^6 = (\sin^2 + \cos^2)^3 - 3\sin^2\cos^2(\sin^2 + \cos^2) = 1 - 3\sin^2 x \cos^2 x \). Sa \( \sin^2 \cos^2 = \tfrac{4}{25} \) dobijamo \( 1 - \tfrac{12}{25} = \tfrac{13}{25} \). Opšte: \( \sin^2 x \cos^2 x = \dfrac{\operatorname{tg}^2 x}{(1+\operatorname{tg}^2 x)^2} \).
