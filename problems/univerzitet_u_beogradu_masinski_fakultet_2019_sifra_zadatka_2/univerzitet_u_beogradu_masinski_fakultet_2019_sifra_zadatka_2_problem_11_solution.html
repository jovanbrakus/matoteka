Trigonometrijski identitet sa zbirom kubova

Trigonometrija — faktorisanje zbira kubova i identitet \( \sin 2x = 2 \sin x \cos x \)

📋 Tekst zadatka

Izraz

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} \]

identički je jednak:

(A) \( \dfrac{\cos x - \sin x}{2} \)
(B) \( 1 \)
(C) \( \dfrac{\cos x}{2} \)
(D) \( \dfrac{1 + \cos^2 x}{4} \)
(E) \( \dfrac{\sin x + \cos x}{2} \)

🎯 Plan rešavanja

Brojilac je zbir kubova \( \sin^3 x + \cos^3 x \), pa ga faktorišemo formulom \( a^3 + b^3 = (a+b)(a^2 - ab + b^2) \) i koristimo \( \sin^2 x + \cos^2 x = 1 \). Imenilac sredimo preko identiteta \( \sin 2x = 2\sin x \cos x \) i izvučemo \( 2 \). Posle skraćivanja zajedničkog faktora ostaje \( (\sin x + \cos x)/2 \).

📚 Podsetnik iz teorije

Zbir kubova i osnovni trigonometrijski identiteti
  • Zbir kubova:
    \[ a^3 + b^3 = (a + b)\,(a^2 - ab + b^2). \]
  • Pitagorin identitet:
    \[ \sin^2 x + \cos^2 x = 1. \]
  • Sinus dvostrukog ugla:
    \[ \sin 2x = 2 \sin x \cos x. \]
  • Granice za \( 2 - \sin 2x \): kako je \( -1 \le \sin 2x \le 1 \), važi \( 1 \le 2 - \sin 2x \le 3 \), pa je imenilac uvek pozitivan i izraz je definisan za sve \( x \in \mathbb{R} \).
  • Korisno: ako je \( a = \sin x,\ b = \cos x \), onda je
    \[ a^2 - ab + b^2 = (a^2 + b^2) - ab = 1 - \sin x \cos x. \]

📐 Vizuelni prikaz — grafici se poklapaju

Zelena puna kriva je vrednost izraza \( \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} \), a narandžasta isprekidana je \( \dfrac{\sin x + \cos x}{2} \). Grafici su identični na celom prikazanom intervalu — to je vizuelna potvrda da se izraz iz zadatka u potpunosti svodi na opciju (E).

✏️ Rešenje korak po korak

1

Faktoriši brojilac kao zbir kubova

Sa \( a = \sin x \) i \( b = \cos x \) primeni formulu \( a^3 + b^3 = (a+b)(a^2 - ab + b^2) \):

\[ \sin^3 x + \cos^3 x = (\sin x + \cos x)\,\big(\sin^2 x - \sin x \cos x + \cos^2 x\big). \]

Drugi činilac lepo se sredi pomoću \( \sin^2 x + \cos^2 x = 1 \):

\[ \sin^2 x - \sin x \cos x + \cos^2 x = 1 - \sin x \cos x. \]

Brojilac je sada

\[ \sin^3 x + \cos^3 x = (\sin x + \cos x)\,(1 - \sin x \cos x). \]
Proveri sebe: zašto \( a^2 + b^2 = 1 \) ovde radi posao?

Imamo \( a = \sin x \) i \( b = \cos x \), pa je \( a^2 + b^2 = \sin^2 x + \cos^2 x = 1 \). U izrazu \( a^2 - ab + b^2 \) jednostavno zamenimo \( a^2 + b^2 \) sa \( 1 \) i ostane samo \( 1 - ab = 1 - \sin x \cos x \).

2

Sredi imenilac preko \( \sin 2x \)

Iskoristimo identitet \( \sin 2x = 2 \sin x \cos x \):

\[ 2 - \sin 2x = 2 - 2 \sin x \cos x = 2\,(1 - \sin x \cos x). \]

U brojiocu i imeniocu pojavljuje se isti činilac \( (1 - \sin x \cos x) \) — to je znak da će se on skratiti.

3

Skrati zajednički činilac

Stavi sve zajedno:

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{(\sin x + \cos x)\,(1 - \sin x \cos x)}{2\,(1 - \sin x \cos x)}. \]

Pošto je \( -1 \le \sin 2x \le 1 \), imenilac \( 2 - \sin 2x \in [1, 3] \) nikad nije \( 0 \). Specijalno, \( 1 - \sin x \cos x = \tfrac{1}{2}(2 - \sin 2x) > 0 \), pa je skraćivanje legalno za svako \( x \in \mathbb{R} \). Ostaje:

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{\sin x + \cos x}{2}. \]
Proveri sebe: kada bismo morali da brinemo za skraćivanje?

Skraćivanje je nedozvoljeno samo kada zajednički činilac može biti \( 0 \) za neku vrednost \( x \). Ovde je \( 1 - \sin x \cos x \ge 1 - \tfrac{1}{2} = \tfrac{1}{2} > 0 \) (jer je \( |\sin x \cos x| = \tfrac{1}{2}|\sin 2x| \le \tfrac{1}{2} \)), pa nema rizika.

4

Numerička provera za \( x = \tfrac{\pi}{4} \)

Brzo proveravamo da nismo pogrešili. Za \( x = \tfrac{\pi}{4} \) je \( \sin x = \cos x = \tfrac{\sqrt 2}{2} \), pa

\[ \sin^3 x + \cos^3 x = 2 \cdot \left(\tfrac{\sqrt 2}{2}\right)^3 = 2 \cdot \tfrac{2\sqrt 2}{8} = \tfrac{\sqrt 2}{2}, \]
\[ 2 - \sin 2x = 2 - \sin\!\tfrac{\pi}{2} = 2 - 1 = 1, \]

pa je

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{\sqrt 2/2}{1} = \dfrac{\sqrt 2}{2}. \]

Sa druge strane, \( \dfrac{\sin x + \cos x}{2} = \dfrac{\sqrt 2/2 + \sqrt 2/2}{2} = \dfrac{\sqrt 2}{2}. \) Slaže se. \( \checkmark \)

💡 Ključni uvid

Kad god u izrazu vidiš zbir kubova \( \sin^3 x + \cos^3 x \), prvi refleks treba da bude faktorisanje \( a^3 + b^3 = (a+b)(a^2 - ab + b^2) \) i upotreba \( \sin^2 x + \cos^2 x = 1 \). Tako se činilac \( a^2 - ab + b^2 \) elegantno svodi na \( 1 - \tfrac{1}{2}\sin 2x \), što obično deli neki član u imeniocu.

Ovde je „magija” u tome što imenilac \( 2 - \sin 2x \) već ima ugrađeno \( 2 \sin x \cos x \) — pa se isti činilac koji se pojavi u brojiocu pojavi i u imeniocu i prirodno se skrati.

⚠️ Česte greške

  • Pisanje \( a^3 + b^3 = (a+b)^3 \). To je pogrešno: pravi razvoj je \( (a+b)^3 = a^3 + 3a^2 b + 3 a b^2 + b^3 \). Tačna formula za zbir kubova je \( (a+b)(a^2 - ab + b^2) \).
  • Zaboravljanje da je \( \sin 2x = 2\sin x\cos x \) (a ne \( 2\sin x + 2\cos x \) ili \( \sin x \cos x \)). Bez ovog identiteta ne vidi se zajednički činilac \( (1 - \sin x \cos x) \).
  • Brkanje \( a^2 - ab + b^2 \) sa \( a^2 - 2ab + b^2 = (a-b)^2 \). Zbir kubova ima samo \( -ab \), a ne \( -2ab \).
  • Skraćivanje bez provere da činilac nije 0. Ovde je sve OK jer je \( 2 - \sin 2x \ge 1 > 0 \), ali u opštem slučaju to treba kratko opravdati.

✅ Konačan odgovor

\[ \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{\sin x + \cos x}{2}. \]

(A) \( \dfrac{\cos x - \sin x}{2} \) (B) \( 1 \) (C) \( \dfrac{\cos x}{2} \) (D) \( \dfrac{1 + \cos^2 x}{4} \) (E) \( \dfrac{\sin x + \cos x}{2} \)

Provera za \( x = \tfrac{\pi}{4} \): obe strane daju \( \tfrac{\sqrt 2}{2} \). Provera za \( x = 0 \): leva strana \( 1/2 \), desna \( (0+1)/2 = 1/2 \). \( \checkmark \) Tačan odgovor je (E).

🏆 Dodatni izazov

Pojednostaviti analognim postupkom:

\[ \dfrac{\sin^3 x - \cos^3 x}{1 - \sin x \cos x}. \]
Pogledaj nagoveštaj

Iskoristi razliku kubova:

\[ a^3 - b^3 = (a-b)(a^2 + ab + b^2). \]

Sa \( a = \sin x,\ b = \cos x \) drugi činilac postaje \( \sin^2 x + \sin x \cos x + \cos^2 x = 1 + \sin x \cos x \). Brojilac je \( (\sin x - \cos x)(1 + \sin x \cos x) \).

Sa imeniocem \( 1 - \sin x \cos x \) dobijamo

\[ \dfrac{(\sin x - \cos x)(1 + \sin x \cos x)}{1 - \sin x \cos x}. \]

Ovde se faktor \( (1 \pm \sin x \cos x) \) ne skraćuje sa imeniocem (znakovi se razlikuju), ali se izraz može zapisati pomoću \( \sin 2x \) — pošto je \( 1 \pm \sin x \cos x = 1 \pm \tfrac{1}{2} \sin 2x \), izraz se svodi na

\[ (\sin x - \cos x) \cdot \dfrac{2 + \sin 2x}{2 - \sin 2x}. \]

Zaključak: ovaj izraz ne ide tako lepo kao originalni — što je dodatan razlog da se cenimo elegantnost glavnog zadatka, gde su znakovi „nameštali”.
