Znakovi koeficijenata kvadratne funkcije iz grafika

Algebra — analiza grafika kvadratne funkcije \(f(x) = ax^2 + bx + c\)

📋 Tekst zadatka

Grafik funkcije \(f(x) = ax^2 + bx + c\) prikazan je na slici. Na grafiku se vidi parabola koja se otvara naviše, čije teme je u četvrtom kvadrantu i koja seče \(y\)-osu ispod koordinatnog početka. Tačan je iskaz:

(A) \(a > 0,\; b > 0,\; c < 0\)
(B) \(a > 0,\; b > 0,\; c > 0\)
(C) \(a > 0,\; b < 0,\; c < 0\)
(D) \(a > 0,\; b < 0,\; c > 0\)
(E) \(a < 0,\; b < 0,\; c < 0\)

🎯 Plan rešavanja

Iz oblika grafika direktno očitavamo znakove koeficijenata: smer otvaranja parabole daje znak \(a\), položaj ose simetrije daje znak \(b\), a presek sa \(y\)-osom daje znak \(c\).

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Smer otvaranja Ako \(a > 0\), parabola se otvara naviše; ako \(a < 0\), naniže.
  • Osa simetrije \(x = -\dfrac{b}{2a}\). Ako je osa desno od \(y\)-ose, onda \(-\dfrac{b}{2a} > 0\).
  • Presek sa \(y\)-osom Tačka \((0, c)\) jer \(f(0) = c\). Znak \(c\) se direktno čita sa grafika.

📐 Vizuelni prikaz

Parabola \(f(x) = ax^2 + bx + c\) sa \(a > 0\), \(b < 0\), \(c < 0\). Teme je u četvrtom kvadrantu, a grafik seče \(y\)-osu ispod koordinatnog početka.

✏️ Rešenje korak po korak

1

Određujemo znak koeficijenta \(a\)

Sa grafika vidimo da se parabola otvara naviše — grane idu ka gore. To neposredno znači:

\[ a > 0 \]

Ovim eliminišemo opciju (E) gde je \(a < 0\).

2

Određujemo znak koeficijenta \(b\)

Teme parabole se nalazi desno od \(y\)-ose, tj. u oblasti \(x > 0\). Osa simetrije parabole je:

\[ x = -\frac{b}{2a} > 0 \]

Pošto je \(a > 0\) (pa je i \(2a > 0\)), da bi razlomak bio pozitivan, brojilac \(-b\) mora biti pozitivan:

\[ -b > 0 \implies b < 0 \]

Ovim eliminišemo opcije (A) i (B) gde je \(b > 0\).

Proveri sebe: Kako se iz položaja temena određuje znak \(b\)?

Teme je u tački \(x_t = -\frac{b}{2a}\). Ako je \(a > 0\) i teme desno od \(y\)-ose (\(x_t > 0\)), onda mora \(b < 0\). Ako je teme levo (\(x_t < 0\)), onda \(b > 0\).

3

Određujemo znak koeficijenta \(c\)

Koeficijent \(c\) predstavlja vrednost funkcije u nuli, tj. \(y\)-odsečak:

\[ f(0) = a \cdot 0^2 + b \cdot 0 + c = c \]

Sa grafika vidimo da parabola seče \(y\)-osu ispod koordinatnog početka, dakle:

\[ c < 0 \]

Ovim eliminišemo opciju (D) gde je \(c > 0\). Ostaje jedino opcija (C).

Proveri sebe: Šta bi značilo \(c = 0\)?

Ako bi \(c = 0\), parabola bi prolazila kroz koordinatni početak \((0, 0)\). To nije slučaj na našem grafiku jer parabola seče \(y\)-osu ispod nule.

4

Dodatna provera — analiza korenova

Sa \(a > 0\), \(b < 0\), \(c < 0\) koristimo Vijetove formule:

\[ x_1 + x_2 = -\frac{b}{a} > 0 \quad (\text{jer } b < 0, a > 0) \]
\[ x_1 \cdot x_2 = \frac{c}{a} < 0 \quad (\text{jer } c < 0, a > 0) \]

Proizvod korenova je negativan, što znači da su koreni različitog znaka — jedan negativan, jedan pozitivan. Zbir je pozitivan, pa je pozitivni koren veći po apsolutnoj vrednosti. Ovo se slaže sa grafikom na kome parabola seče \(x\)-osu na dva mesta.

💡 Ključni uvid

Tri koeficijenta kvadratne funkcije se mogu „pročitati“ direktno sa grafika pomoću tri jednostavna pravila: \(a\) → smer otvaranja, \(b\) → položaj ose simetrije u odnosu na \(y\)-osu (uz poznati znak \(a\)), \(c\) → presek sa \(y\)-osom. Ne treba nam nijedna numerička vrednost — dovoljni su samo znakovi!

✅ Konačan odgovor

\[ a > 0, \quad b < 0, \quad c < 0 \]

(A) \(a > 0,\; b > 0,\; c < 0\) (B) \(a > 0,\; b > 0,\; c > 0\) (C) \(a > 0,\; b < 0,\; c < 0\) (D) \(a > 0,\; b < 0,\; c > 0\) (E) \(a < 0,\; b < 0,\; c < 0\)

Provera: Za primer \(f(x) = x^2 - 3x - 2\) imamo \(a = 1 > 0\), \(b = -3 < 0\), \(c = -2 < 0\). Teme je u tački \(x = \frac{3}{2} > 0\), \(f(0) = -2 < 0\). Slaže se sa grafikom. ✓

⚠️ Česte greške

  • Brkanje znaka \(b\) sa položajem temena: Ako je teme desno od \(y\)-ose i \(a > 0\), onda je \(b < 0\) (ne \(b > 0\)). Formula \(x_t = -\frac{b}{2a}\) ima minus ispred \(b\)!
  • Zaboravljanje da je \(c\) presek sa \(y\)-osom: Dovoljno je pogledati gde grafik seče \(y\)-osu. Ako je ispod nule, \(c < 0\).
  • Miješanje smera otvaranja: Parabola naviše ↔ \(a > 0\), naniže ↔ \(a < 0\). Ovo je najdirektnija informacija sa grafika.

🏆 Dodatni izazov

Ako parabola \(f(x) = ax^2 + bx + c\) ima teme u tački \((2, -3)\) i prolazi kroz tačku \((0, -1)\), odredi koeficijente \(a\), \(b\) i \(c\) i proveri da znakovi odgovaraju našoj analizi.

Pogledaj nagoveštaj

Koristi kanonski oblik: \(f(x) = a(x - 2)^2 - 3\). Iz uslova \(f(0) = -1\) dobij \(a(0-2)^2 - 3 = -1\), tj. \(4a = 2\), pa \(a = \frac{1}{2}\). Zatim razvij u standardni oblik da nađeš \(b\) i \(c\).
