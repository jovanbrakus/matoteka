Nule proizvoda funkcija i unija skupova

Teorija skupova, nule funkcija, osobine realnih brojeva

Tekst zadatka

Neka su \( f, g : \mathbb{R} \to \mathbb{R} \) funkcije i neka je \( A \) skup svih realnih brojeva \( x \) za koje je \( f(x) = 0 \), \( B \) skup svih realnih brojeva \( x \) za koje je \( g(x) = 0 \) i \( C \) skup svih realnih brojeva \( x \) za koje je \( f(x) \cdot g(x) = 0 \). Tada uvek važi:

(A) \( C = A \cap B \)
(B) \( C = A \cup B \)
(C) \( C = A \setminus B \)
(D) \( C = B \setminus A \)
(E) \( C = A \times B \)

Plan rešavanja

Iskoristićemo fundamentalnu osobinu realnih brojeva da je proizvod dva broja nula ako i samo ako je bar jedan od njih nula, a zatim ćemo to prevesti na jezik teorije skupova.

Podsetnik iz teorije

Ključne formule i teoreme
  • Osobina nultog proizvoda: Za \( a, b \in \mathbb{R} \): \( a \cdot b = 0 \iff a = 0 \) ili \( b = 0 \).
  • Unija skupova: \( A \cup B = \{x : x \in A \text{ ili } x \in B\} \)
  • Presek skupova: \( A \cap B = \{x : x \in A \text{ i } x \in B\} \)
  • Razlika skupova: \( A \setminus B = \{x : x \in A \text{ i } x \notin B\} \)

Vizuelni prikaz

Venov dijagram skupova A (nule funkcije f) i B (nule funkcije g). Skup C = A \u222a B obuhvata celu obojenu oblast jer je f(x)\u00b7g(x) = 0 kad god je bar jedan faktor nula.

Rešenje korak po korak

1

Precizno definišemo skupove

Zapišimo skupove koristeći matematičku notaciju:

\[ \begin{align} A &= \{x \in \mathbb{R} : f(x) = 0\} \\[4pt] B &= \{x \in \mathbb{R} : g(x) = 0\} \\[4pt] C &= \{x \in \mathbb{R} : f(x) \cdot g(x) = 0\} \end{align} \]

Skupovi \( A \) i \( B \) su skupovi nula (korenova) funkcija \( f \) i \( g \), a \( C \) je skup nula njihovog proizvoda.

2

Primenimo osobinu nultog proizvoda

Za realne brojeve važi fundamentalna osobina:

\( a \cdot b = 0 \iff a = 0 \) ili \( b = 0 \)

Primenimo ovo na naš slučaj. Element \( x \) pripada skupu \( C \) ako i samo ako:

\[ f(x) \cdot g(x) = 0 \iff f(x) = 0 \text{ ili } g(x) = 0 \]
Proveri sebe: Zašto važi osobina nultog proizvoda?

Jer su realni brojevi integralni domen — nemaju delitelje nule. Ako je \( a \neq 0 \) i \( b \neq 0 \), onda je i \( a \cdot b \neq 0 \). Kontraponiranjem: ako je \( a \cdot b = 0 \), onda je \( a = 0 \) ili \( b = 0 \).

3

Prevedemo na jezik skupova

Uslov „\( f(x) = 0 \) ili \( g(x) = 0 \)" u jeziku skupova znači:

\[ x \in A \text{ ili } x \in B \iff x \in A \cup B \]

Dakle:

\[ C = A \cup B \]
Proveri sebe: Zašto „ili" odgovara uniji, a ne preseku?

Unija \( A \cup B \) sadrži elemente koji pripadaju barem jednom od skupova, što odgovara logičkom „ili". Presek \( A \cap B \) sadrži elemente koji pripadaju obema skupovima istovremeno, što odgovara logičkom „i".

4

Zašto ostali odgovori nisu tačni

Razmotrimo konkretan primer: neka je \( f(x) = x \) i \( g(x) = x - 1 \). Tada:

\[ A = \{0\}, \quad B = \{1\}, \quad C = \{0,\; 1\} \]
  • \( A \cap B = \emptyset \neq C \) — presek je prazan, ali \( C \) ima dva elementa.
  • \( A \setminus B = \{0\} \neq C \) — nedostaje element 1.
  • \( B \setminus A = \{1\} \neq C \) — nedostaje element 0.
  • \( A \times B = \{(0, 1)\} \) — ovo je skup uređenih parova, a \( C \) je skup brojeva. Pogrešan tip objekta.
Proveri sebe: Kada bi bio tačan odgovor A ∩ B?

Presek \( A \cap B \) je skup onih \( x \) za koje važi \( f(x) = 0 \) i \( g(x) = 0 \) istovremeno. To bi bio tačan odgovor za pitanje „za koja \( x \) važi \( f(x)^2 + g(x)^2 = 0 \)?" jer zbir kvadrata realnih brojeva je nula samo ako su oba nula.

Ključni uvid

Ovaj zadatak povezuje dva fundamentalna koncepta: osobinu nultog proizvoda u algebri i operaciju unije u teoriji skupova. Logičko „ili" (disjunkcija) direktno odgovara skupovnoj operaciji unije (\( \cup \)), dok logičko „i" (konjunkcija) odgovara preseku (\( \cap \)). Ova veza između logike i teorije skupova je jedan od najvažnijih mostova u matematici.

Konačan odgovor

\[ C = A \cup B \]

(A) \( C = A \cap B \) (B) \( C = A \cup B \) (C) \( C = A \setminus B \) (D) \( C = B \setminus A \) (E) \( C = A \times B \)

Verifikacija: Za \( f(x) = x \), \( g(x) = x - 1 \): \( A = \{0\} \), \( B = \{1\} \), \( C = \{0, 1\} = A \cup B \). Za \( f(x) = x(x-1) \), \( g(x) = (x-1)(x-2) \): \( A = \{0, 1\} \), \( B = \{1, 2\} \), \( C = \{0, 1, 2\} = A \cup B \). Tačno u oba slučaja.

Česte greške

  • Mešanje unije i preseka: Učenici često pomešaju „ili" sa „i". Proizvod je nula kad je bar jedan faktor nula (unija), a ne kad su oba nula (presek).
  • Zaboravljanje jednog smera ekvivalencije: Treba pokazati i da \( C \subseteq A \cup B \) i da \( A \cup B \subseteq C \). Osobina nultog proizvoda je ekvivalencija (\( \iff \)), pa važe oba smera.
  • Brkanje \( A \times B \) sa ostalim operacijama: Dekartov proizvod \( A \times B \) daje skup uređenih parova, što je potpuno drugačija vrsta objekta od \( C \) koji je podskup od \( \mathbb{R} \).

Dodatni izazov

Ako umesto \( f(x) \cdot g(x) = 0 \) posmatramo \( f(x)^2 + g(x)^2 = 0 \), koji od ponuđenih odgovora bi bio tačan? A šta je sa uslovom \( \max(|f(x)|, |g(x)|) = 0 \)?

Pogledaj nagoveštaj

Za \( f(x)^2 + g(x)^2 = 0 \): pošto su oba sabirka nenegativna, njihov zbir je nula samo ako su oba nula, tj. \( f(x) = 0 \) i \( g(x) = 0 \). Dakle, skup rešenja je \( A \cap B \) (presek). Za \( \max(|f(x)|, |g(x)|) = 0 \): maksimum je nula samo ako su oba argumenta nula, pa je opet \( A \cap B \). Obratite pažnju: „i" daje presek, a „ili" daje uniju!
