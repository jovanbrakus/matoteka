Predznaci koeficijenata kvadratne funkcije iz grafika

Kvadratna funkcija — veza između koeficijenata i grafika parabole

📋 Tekst zadatka

Grafik funkcije \( f(x) = ax^2 + bx + c \) prikazan je na slici. Parabola se otvara naviše, teme joj se nalazi u četvrtom kvadrantu (desno od \( y \)-ose, ispod \( x \)-ose), a presek sa \( y \)-osom je ispod \( x \)-ose. Tačan je iskaz:

(A) \( a > 0,\; b > 0,\; c < 0 \)
(B) \( a > 0,\; b > 0,\; c > 0 \)
(C) \( a > 0,\; b < 0,\; c < 0 \)
(D) \( a > 0,\; b < 0,\; c > 0 \)
(E) \( a < 0,\; b < 0,\; c < 0 \)

🎯 Plan rešavanja

Iz izgleda grafika parabole očitaćemo tri ključne informacije: smer otvaranja (znak \( a \)), položaj temena (znak \( b \)), i presek sa \( y \)-osom (znak \( c \)).

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Smer otvaranja: Ako je \( a > 0 \), parabola se otvara naviše (U-oblik). Ako je \( a < 0 \), otvara se naniže (∩-oblik).
  • Teme parabole: \( x \)-koordinata temena je \( x_t = -\dfrac{b}{2a} \). Ako je teme desno od \( y \)-ose, tada \( x_t > 0 \).
  • Presek sa \( y \)-osom: \( f(0) = c \). Ako je presek ispod \( x \)-ose, tada \( c < 0 \).
  • Vjetove formule: Za korene \( x_1 \) i \( x_2 \): \( x_1 + x_2 = -\frac{b}{a} \) i \( x_1 \cdot x_2 = \frac{c}{a} \).

📐 Vizuelni prikaz

Grafik funkcije \( f(x) = ax^2 + bx + c \) sa označenim ključnim osobinama: smer otvaranja, položaj temena, presek sa \( y \)-osom i nule funkcije

✏️ Rešenje korak po korak

1

Odredimo znak koeficijenta \( a \)

Parabola se otvara naviše (ima oblik slova U). To znači:

\[ a > 0 \]

Ovo odmah eliminše odgovor (E), jer tamo stoji \( a < 0 \).

Proveri sebe: Kako bismo prepoznali \( a < 0 \) na grafiku?

Parabola bi se otvarala naniže (oblik slova ∩). Grafik bi imao maksimum umesto minimuma.

2

Odredimo znak koeficijenta \( b \)

Teme parabole se nalazi u četvrtom kvadrantu, dakle desno od \( y \)-ose. To znači da je \( x \)-koordinata temena pozitivna:

\[ x_t = -\frac{b}{2a} > 0 \]

Pošto znamo da je \( a > 0 \), imenilac \( 2a \) je pozitivan. Da bi ceo razlomak bio pozitivan, potrebno je da je i brojilac pozitivan:

\[ -b > 0 \quad \Rightarrow \quad b < 0 \]

Ovo eliminše odgovore (A) i (B), jer tamo stoji \( b > 0 \).

Proveri sebe: Šta bi značilo \( b > 0 \) za položaj temena?

Uz \( a > 0 \) i \( b > 0 \), imali bismo \( x_t = -\frac{b}{2a} < 0 \), što znači da bi teme bilo levo od \( y \)-ose (u drugom ili trećem kvadrantu).

3

Odredimo znak koeficijenta \( c \)

Presek parabole sa \( y \)-osom dobijamo za \( x = 0 \):

\[ f(0) = a \cdot 0^2 + b \cdot 0 + c = c \]

Grafik seče \( y \)-osu ispod \( x \)-ose, što znači:

\[ c < 0 \]

Ovo eliminše odgovor (D), jer tamo stoji \( c > 0 \). Ostaje samo odgovor (C).

4

Verifikacija pomoću Vjetovih formula

Možemo dodatno proveriti konzistentnost. Grafik seče \( x \)-osu u dve tačke — funkcija ima dve realne nule \( x_1 \) i \( x_2 \).

Pošto je \( c < 0 \) i \( a > 0 \):

\[ x_1 \cdot x_2 = \frac{c}{a} < 0 \]

Proizvod korena je negativan, što znači da koreni imaju suprotne znake — jedan je negativan, drugi pozitivan. Ovo se slaže sa grafikom na kome parabola seče \( x \)-osu na po jednom mestu sa svake strane koordinatnog početka.

Proveri sebe: Zašto diskriminanta mora biti pozitivna?

\( D = b^2 - 4ac \). Pošto je \( a > 0 \) i \( c < 0 \), imamo \( -4ac > 0 \), pa je \( D = b^2 + |4ac| > 0 \). Grafik zaista seče \( x \)-osu u dve tačke.

💡 Ključni uvid

Svaki koeficijent kvadratne funkcije \( f(x) = ax^2 + bx + c \) ima jasnu geometrijsku interpretaciju na grafiku:

  • \( a \) — kontrolše smer otvaranja (naviše/naniže)
  • \( b \) — zajedno sa \( a \) određuje položaj temena (levo/desno od \( y \)-ose)
  • \( c \) — određuje presek sa \( y \)-osom (iznad/ispod \( x \)-ose)

Ova „čitalačka tehnika" je brza i pouzdana — ne zahteva nikakvo računanje!

✅ Konačan odgovor

\[ a > 0, \quad b < 0, \quad c < 0 \]

(A) \( a > 0,\; b > 0,\; c < 0 \) (B) \( a > 0,\; b > 0,\; c > 0 \) (C) \( a > 0,\; b < 0,\; c < 0 \) (D) \( a > 0,\; b < 0,\; c > 0 \) (E) \( a < 0,\; b < 0,\; c < 0 \)

Verifikacija: Primer funkcije \( f(x) = x^2 - 2x - 3 = (x - 3)(x + 1) \) sa \( a = 1 > 0 \), \( b = -2 < 0 \), \( c = -3 < 0 \) ima teme u tački \( (1, -4) \) (četvrti kvadrant), presek sa \( y \)-osom u \( (0, -3) \) (ispod \( x \)-ose), i otvara se naviše — sve se slaže sa opisom grafika. ✓

⚠️ Česte greške

  • Pogrešan zaključak o znaku \( b \) iz položaja temena: Čest je lapsus da se iz \( x_t > 0 \) zaključi \( b > 0 \). Ali formula je \( x_t = -b/(2a) \), pa je potrebno voditi računa o minusu!
  • Brkanje \( c \) sa vrednošću temena: Koeficijent \( c \) je presek sa \( y \)-osom (\( f(0) = c \)), a ne \( y \)-koordinata temena.
  • Izbor (D) umesto (C): Odgovor (D) ima \( c > 0 \), što bi značilo da grafik seče \( y \)-osu iznad \( x \)-ose — suprotno od onoga što vidimo na grafiku.

🏆 Dodatni izazov

Ako je teme parabole u drugom kvadrantu (levo od \( y \)-ose, iznad \( x \)-ose), a presek sa \( y \)-osom je iznad \( x \)-ose, koji bi predznaci koeficijenata bili? Razmisli koje od ponuđenih opcija bi to odgovaralo.

Pogledaj nagoveštaj

Teme u drugom kvadrantu znači \( x_t = -b/(2a) < 0 \). Ako parabola i dalje otvara naviše (\( a > 0 \)), tada je \( -b < 0 \), tj. \( b > 0 \).

Presek sa \( y \)-osom iznad \( x \)-ose znači \( c > 0 \).

Dakle: \( a > 0 \), \( b > 0 \), \( c > 0 \) — to bi bio odgovor (B).

Primer: \( f(x) = x^2 + 2x + 3 \), teme \( (-1, 2) \), \( f(0) = 3 > 0 \).
