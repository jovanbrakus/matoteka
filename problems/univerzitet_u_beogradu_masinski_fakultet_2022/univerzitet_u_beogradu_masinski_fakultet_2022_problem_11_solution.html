Pojednostavljivanje izraza \( \dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} \)

Trigonometrija — zbir kubova, formula za \( \sin 2x \) i osnovni Pitagorin identitet \( \sin^2 x + \cos^2 x = 1 \).

📋 Tekst zadatka

Izraz

\[ E(x) \;=\; \dfrac{\sin^{3} x \;+\; \cos^{3} x}{2 \;-\; \sin 2x} \]

identički je jednak:

(A) \( \dfrac{\cos x - \sin x}{2} \)
(B) \( 1 \)
(C) \( \dfrac{\cos x}{2} \)
(D) \( \dfrac{1 + \cos^{2} x}{4} \)
(E) \( \dfrac{\sin x + \cos x}{2} \)

🎯 Plan rešavanja

Brojilac je zbir kubova \( \sin^3 x + \cos^3 x \) — faktorišemo ga i iskoristimo \( \sin^2 x + \cos^2 x = 1 \), čime ispada zajednički činilac \( (1 - \sin x \cos x) \). Imenilac iskoristimo preko \( \sin 2x = 2 \sin x \cos x \), pa \( 2 - \sin 2x = 2(1 - \sin x \cos x) \). Skraćivanjem ostaje \( \tfrac{\sin x + \cos x}{2} \).

📚 Podsetnik iz teorije

Zbir kubova, dvostruki ugao i osnovni identitet
  • Zbir kubova: za sve realne (ili kompleksne) brojeve \( A, B \):
    \[ A^{3} + B^{3} \;=\; (A + B)\,(A^{2} - A B + B^{2}). \]
  • Osnovni Pitagorin identitet: \( \sin^{2} x + \cos^{2} x = 1 \) za sve \( x \).
  • Sinus dvostrukog ugla:
    \[ \sin 2x \;=\; 2 \sin x \cos x. \]
  • Skraćivanje u racionalnom izrazu: deljenje brojioca i imenioca istim faktorom dozvoljeno je samo ako taj faktor nije \( 0 \) za posmatrane vrednosti promenljive. Ovde će se pokazati da je \( 1 - \sin x \cos x \) uvek strogo pozitivno.

✏️ Rešenje korak po korak

1

Faktorisanje brojioca: zbir kubova

Iskoristimo formulu za zbir kubova sa \( A = \sin x \) i \( B = \cos x \):

\[ \sin^{3} x + \cos^{3} x \;=\; (\sin x + \cos x)\,\bigl( \sin^{2} x - \sin x \cos x + \cos^{2} x \bigr). \]

Druga zagrada se pojednostavljuje koristeći \( \sin^{2} x + \cos^{2} x = 1 \):

\[ \sin^{2} x - \sin x \cos x + \cos^{2} x \;=\; \underbrace{(\sin^{2} x + \cos^{2} x)}_{= 1} - \sin x \cos x \;=\; 1 - \sin x \cos x. \]

Dakle:

\[ \sin^{3} x + \cos^{3} x \;=\; (\sin x + \cos x)\,(1 - \sin x \cos x). \tag{*} \]
Proveri sebe: zašto \( (\sin x + \cos x)^{3} \neq \sin^{3} x + \cos^{3} x \)?

Razvijanjem kuba zbira: \( (a+b)^{3} = a^{3} + 3 a^{2} b + 3 a b^{2} + b^{3} = a^{3} + b^{3} + 3 a b (a + b) \). Mešovitih članova ima — kub zbira nije zbir kubova osim u trivijalnom slučaju \( a b (a+b) = 0 \). Zato koristimo formulu za zbir kubova, a ne kub zbira.

2

Imenilac preko \( \sin 2x \)

Iz formule za sinus dvostrukog ugla \( \sin 2x = 2 \sin x \cos x \):

\[ 2 - \sin 2x \;=\; 2 - 2 \sin x \cos x \;=\; 2\,(1 - \sin x \cos x). \tag{**} \]

Imenilac sada izgleda vrlo slično zagradi koju smo dobili u (*).

3

Skraćivanje zajedničkog faktora

Ubacimo (*) i (**) u izraz:

\[ E(x) \;=\; \dfrac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \;=\; \dfrac{(\sin x + \cos x)\,\bigl(1 - \sin x \cos x\bigr)}{2\,\bigl(1 - \sin x \cos x\bigr)}. \]

Brojilac i imenilac dele faktor \( (1 - \sin x \cos x) \). Ako je on različit od nule, možemo ga skratiti:

\[ E(x) \;=\; \dfrac{\sin x + \cos x}{2}. \]
4

Provera dozvoljenosti skraćivanja

Treba pokazati da \( 1 - \sin x \cos x \neq 0 \) za sve realne \( x \). Iskoristimo \( \sin x \cos x = \tfrac{1}{2}\sin 2x \):

\[ 1 - \sin x \cos x \;=\; 1 - \tfrac{1}{2}\sin 2x. \]

Pošto je \( -1 \le \sin 2x \le 1 \), sledi \( -\tfrac{1}{2} \le \tfrac{1}{2}\sin 2x \le \tfrac{1}{2} \), pa

\[ \tfrac{1}{2} \;\le\; 1 - \sin x \cos x \;\le\; \tfrac{3}{2}. \]

Vrednost je uvek najmanje \( \tfrac{1}{2} \), dakle strogo pozitivna i različita od nule. Skraćivanje je u redu za svako \( x \in \mathbb{R} \), pa identitet važi univerzalno.

Proveri sebe: kada se \( 1 - \sin x \cos x \) najviše približi nuli?

Najmanja vrednost \( 1 - \tfrac{1}{2}\sin 2x \) se postiže kad je \( \sin 2x = 1 \), tj. \( 2x = \tfrac{\pi}{2} + 2k\pi \), odnosno \( x = \tfrac{\pi}{4} + k\pi \). Tada vrednost iznosi \( 1 - \tfrac{1}{2} = \tfrac{1}{2} \). I dalje strogo veće od \( 0 \) — bezbedno za skraćivanje.

5

Numerička provera u tri tačke

Brza provera identiteta na tri zgodne vrednosti pokazuje slaganje LHS = RHS:

  • \( x = 0 \): LHS \( = \dfrac{0 + 1}{2 - 0} = \dfrac{1}{2} \); RHS \( = \dfrac{0 + 1}{2} = \dfrac{1}{2} \). ✓
  • \( x = \tfrac{\pi}{4} \): LHS \( = \dfrac{2\,(\tfrac{\sqrt{2}}{2})^{3}}{2 - 1} = \dfrac{\sqrt{2}/2}{1} = \dfrac{\sqrt{2}}{2} \); RHS \( = \dfrac{\tfrac{\sqrt{2}}{2} + \tfrac{\sqrt{2}}{2}}{2} = \dfrac{\sqrt{2}}{2} \). ✓
  • \( x = \pi \): LHS \( = \dfrac{0 + (-1)}{2 - 0} = -\dfrac{1}{2} \); RHS \( = \dfrac{0 + (-1)}{2} = -\dfrac{1}{2} \). ✓

Ostale ponuđene opcije ne prolaze sve tri provere (vidi karticu „Česte greške" za detalje).

💡 Ključni uvid

Kad god se u trigonometrijskim izrazima pojave kubovi sinusa i kosinusa, faktorisanje preko zbira (ili razlike) kubova obično otkriva krati zajednički faktor sa imeniocem koji sadrži \( \sin 2x \) ili \( \cos 2x \). Ovde je to bio \( (1 - \sin x \cos x) \). Algoritamski recept: faktoriši brojilac, prepoznaj \( \sin 2x = 2 \sin x \cos x \) u imeniocu, skrati. Uvek proveri da je faktor koji se skraćuje različit od nule.

📐 Vizuelni prikaz: dve funkcije se preklapaju

Plot leve strane \( E(x) = \dfrac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \) i desne strane \( \dfrac{\sin x + \cos x}{2} \) na intervalu \( [-\pi, 2\pi] \). Krive se savršeno poklapaju (jedna preko druge), što potvrđuje identitet. Plava (LHS) je iscrtana neprekidno, a zelena (RHS) iscrtkano preko nje — vidi se da se savršeno slažu na svakoj tački.

✅ Konačan odgovor

\[ \dfrac{\sin^{3} x + \cos^{3} x}{2 - \sin 2x} \;=\; \dfrac{\sin x + \cos x}{2} \quad \text{za svako}\ x \in \mathbb{R}. \]

(A) \( \dfrac{\cos x - \sin x}{2} \) (B) \( 1 \) (C) \( \dfrac{\cos x}{2} \) (D) \( \dfrac{1 + \cos^{2} x}{4} \) (E) \( \dfrac{\sin x + \cos x}{2} \)

Provera u \( x = \tfrac{\pi}{4} \): LHS \( = \dfrac{2 (\tfrac{\sqrt{2}}{2})^{3}}{2 - 1} = \dfrac{\sqrt{2}}{2} \), RHS \( = \dfrac{\sqrt{2}}{2} \). ✓ Identitet važi za sve realne \( x \) jer je \( 1 - \sin x \cos x \in [\tfrac{1}{2}, \tfrac{3}{2}] \), pa skraćivanje nikad ne deli sa nulom.

⚠️ Česte greške

  • Pisanje \( \sin^{3} x + \cos^{3} x = (\sin x + \cos x)^{3} \) — pogrešno. Kub zbira ima i mešovite članove: \( (a+b)^{3} = a^{3} + b^{3} + 3 a b (a+b) \).
  • Zaboravljanje faktora \( 2 \) u \( \sin 2x = 2 \sin x \cos x \). Bez tog \( 2 \) ne bi ispao zajednički faktor sa brojiocem.
  • Zaključivanje da je odgovor (D) \( \dfrac{1 + \cos^{2} x}{4} \) jer u \( x = 0 \) obe opcije daju \( \tfrac{1}{2} \). Probom u drugoj tački (npr. \( x = \tfrac{\pi}{2} \)) (D) daje \( \tfrac{1}{4} \), a tačan odgovor (E) daje \( \tfrac{1}{2} \).
  • Pokušaj „uniformizacije" preko \( \cos^{3} x = \cos x (1 - \sin^{2} x) \) i razvijanja — funkcionalan, ali znatno duži put. Faktorisanje zbira kubova daje rezultat u jednom potezu.
  • Skraćivanje bez provere da je \( 1 - \sin x \cos x \neq 0 \). Ovde je dokazano da je strogo pozitivno, ali takvu proveru treba navesti kao posebnu liniju rezonovanja.

\[ F(x) \;=\; \dfrac{\sin^{3} x \;-\; \cos^{3} x}{2 \;+\; \sin 2x}. \]

Pogledaj nagoveštaj

Brojilac je razlika kubova: \( \sin^{3} x - \cos^{3} x = (\sin x - \cos x)(\sin^{2} x + \sin x \cos x + \cos^{2} x) = (\sin x - \cos x)(1 + \sin x \cos x) \). Imenilac: \( 2 + \sin 2x = 2(1 + \sin x \cos x) \). Skraćivanje daje \( F(x) = \dfrac{\sin x - \cos x}{2} \). Faktor \( 1 + \sin x \cos x = 1 + \tfrac{1}{2}\sin 2x \in [\tfrac{1}{2}, \tfrac{3}{2}] \) je opet uvek pozitivan, pa identitet važi za sve \( x \in \mathbb{R} \).
