Broj rešenja trigonometrijske jednačine

Trigonometrija — rešavanje jednačina, faktorisanje, dvostruki ugao

📋 Tekst zadatka

Broj rešenja jednačine

\[\sin x + \dfrac{1}{\sqrt{3}}\,\sin 2x = 0\]

na intervalu \((0, \pi)\) je:

(A) \(0\)
(B) \(1\)
(C) \(2\)
(D) \(3\)
(E) \(4\)

🎯 Plan rešavanja

Iskoristićemo formulu \(\sin 2x = 2\sin x\cos x\) da izvučemo zajednički činilac \(\sin x\). Jednačina se raspada na dva jednostavnija slučaja; preostaje samo da proverimo koja rešenja padaju u otvoreni interval \((0, \pi)\).

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Sinus dvostrukog ugla \(\sin 2x = 2\sin x\cos x\).
  • Nule funkcije sinus \(\sin x = 0 \iff x = k\pi,\; k\in\mathbb{Z}\).
  • Vrednosti kosinusa \(\cos x = -\dfrac{\sqrt{3}}{2} \iff x = \pm \dfrac{5\pi}{6} + 2k\pi,\; k\in\mathbb{Z}\).
  • Otvoren interval \((0, \pi)\) ne sadrži krajnje tačke \(0\) i \(\pi\); moramo biti pažljivi sa graničnim rešenjima.
  • Nula proizvoda \(a\cdot b = 0 \iff a = 0 \;\lor\; b = 0\). Osnovno sredstvo za rastavljanje trig jednačine.

📐 Grafik funkcije

Grafik \(f(x) = \sin x + \tfrac{1}{\sqrt{3}}\sin 2x\) na \((0,\pi)\) — tačno jedna nula u \(x = \tfrac{5\pi}{6}\)

✏️ Rešenje korak po korak

1

Razvijanje \(\sin 2x\)

Primenimo formulu \(\sin 2x = 2\sin x\cos x\):

\[\sin x + \dfrac{1}{\sqrt{3}}\cdot 2\sin x\cos x = 0\]
\[\sin x + \dfrac{2}{\sqrt{3}}\,\sin x\,\cos x = 0\]
2

Izvlačenje zajedničkog činioca

Obe aditivne komponente sadrže \(\sin x\), pa izvlačimo:

\[\sin x\left(1 + \dfrac{2}{\sqrt{3}}\cos x\right) = 0\]

Jednačina se raspada na dva nezavisna slučaja koja analiziramo posebno.

Proveri sebe: zašto sme da se „deli sa \(\sin x\)“ samo kada \(\sin x \ne 0\)?

Deljenje nulom nije dozvoljeno, a pored toga tada bismo izgubili potencijalna rešenja gde je \(\sin x = 0\). Zato pravilno postupamo: pišemo proizvod = 0 i razmatramo oba faktora.

3

Slučaj A: \(\sin x = 0\)

Sva rešenja su \(x = k\pi\), \(k \in \mathbb{Z}\). Od njih nas zanimaju samo ona u otvorenom intervalu \((0, \pi)\):

\[x = 0 \;\text{(izostavljeno)}, \quad x = \pi \;\text{(izostavljeno)}\]

Dakle slučaj A ne daje nijedno rešenje unutar \((0, \pi)\).

4

Slučaj B: \(1 + \dfrac{2}{\sqrt{3}}\cos x = 0\)

Rešavamo za \(\cos x\):

\[\dfrac{2}{\sqrt{3}}\cos x = -1 \;\Longrightarrow\; \cos x = -\dfrac{\sqrt{3}}{2}\]

Opšte rešenje:

\[x = \pm \dfrac{5\pi}{6} + 2k\pi,\quad k \in \mathbb{Z}\]
5

Izbor rešenja iz \((0, \pi)\)

Od svih \(x\) sa \(\cos x = -\tfrac{\sqrt{3}}{2}\) pitamo koja su u \((0, \pi)\):

  • \(x = \tfrac{5\pi}{6}\): jeste, \(0 < \tfrac{5\pi}{6} < \pi\). \(\checkmark\)
  • \(x = -\tfrac{5\pi}{6}\): nije (negativno).
  • \(x = -\tfrac{5\pi}{6} + 2\pi = \tfrac{7\pi}{6}\): nije, veće od \(\pi\).
  • Ostali pomeraji za \(2k\pi\) izvode nas još dalje od \((0, \pi)\).

Dakle slučaj B daje tačno jedno rešenje: \(x = \tfrac{5\pi}{6}\).

Proveri sebe: zašto \(\cos x = -\tfrac{\sqrt{3}}{2}\) u drugom kvadrantu, a ne prvom?

Kosinus je negativan u II i III kvadrantu. Unutar \((0, \pi)\) imamo I i II kvadrant, a samo II (tj. \(\pi/2 < x < \pi\)) daje negativan kosinus. \(\cos\tfrac{5\pi}{6} = -\cos\tfrac{\pi}{6} = -\tfrac{\sqrt{3}}{2}\) \(\checkmark\).

6

Sabiranje slučajeva

Slučaj A: \(0\) rešenja. Slučaj B: \(1\) rešenje. Ukupan broj rešenja u \((0, \pi)\):

\[0 + 1 = 1.\]

Brojčana provera: za \(x = \tfrac{5\pi}{6}\) imamo \(\sin x = \tfrac{1}{2}\) i \(\sin 2x = \sin\tfrac{5\pi}{3} = -\tfrac{\sqrt{3}}{2}\), pa:

\[\sin x + \tfrac{1}{\sqrt{3}}\sin 2x = \tfrac{1}{2} + \tfrac{1}{\sqrt{3}}\cdot\left(-\tfrac{\sqrt{3}}{2}\right) = \tfrac{1}{2} - \tfrac{1}{2} = 0\;\checkmark\]

💡 Ključni uvid

Kad god vidite \(\sin x\) i \(\sin 2x\) (ili \(\cos 2x\)) u istoj jednačini, razvijte \(2x\) formulom dvostrukog ugla da dobijete zajednički činilac \(\sin x\) ili \(\cos x\). Time se problem raspada na nekoliko jednostavnijih jednačina koje rešavamo tablično. Druga ključna tanka linija je tip intervala: ovde je \((0, \pi)\) otvoren, pa \(\sin x = 0\) gubi oba kandidata (\(0\) i \(\pi\)) — da je bilo \([0, \pi]\), rešenja bi bilo tri.

✅ Konačan odgovor

\[\text{Broj rešenja} = 1 \quad\left(x = \tfrac{5\pi}{6}\right)\]

(A) \(0\) (B) \(1\) (C) \(2\) (D) \(3\) (E) \(4\)

Provera: \(\sin\tfrac{5\pi}{6} + \tfrac{1}{\sqrt{3}}\sin\tfrac{5\pi}{3} = \tfrac{1}{2} - \tfrac{1}{2} = 0\) \(\checkmark\). Grafik (\(f(x) = \sin x + \tfrac{1}{\sqrt{3}}\sin 2x\)) preseca \(x\)-osu u \((0,\pi)\) tačno jednom, između \(\tfrac{\pi}{2}\) i \(\pi\) \(\checkmark\).

⚠️ Česte greške

  • Deljenje sa \(\sin x\): ako „uprostite“ jednačinu tako što podelite sa \(\sin x\), gubite rešenja gde je \(\sin x = 0\). Ovde se ispostavi da ne igraju ulogu (jer interval je otvoren), ali u drugom zadatku lako možete izgubiti rešenja — pravilno: faktorišite, ne delite.
  • Otvoren vs. zatvoren interval: na \([0, \pi]\) tačke \(0\) i \(\pi\) bi se uračunale kao rešenja (\(\sin 0 = \sin \pi = 0\)), pa bi ukupno bilo \(3\), što vodi odgovoru (D) — klasična zamka.
  • Pogrešno inverzan kosinus: \(\cos x = -\tfrac{\sqrt{3}}{2}\) NE znači \(x = \tfrac{\pi}{6}\) (to bi bilo \(\cos x = \tfrac{\sqrt{3}}{2}\)). Znak minus vraća u II kvadrant: \(x = \tfrac{5\pi}{6}\).
  • Ignorisanje \((2/\sqrt{3})\) koeficijenta: neki studenti „preskoče“ racionalni koeficijent i dobiju \(\cos x = -1\) (odnosno \(x = \pi\)), ali to je izvan otvorenog intervala i ne zadovoljava originalnu jednačinu.
  • Odgovor (C) dva rešenja: gluvo brojanje \(\pm 5\pi/6\) ili \(5\pi/6\) i \(7\pi/6\) bez provere granica daje \(2\).

🏆 Dodatni izazov

Koliko bi rešenja imala ista jednačina \(\sin x + \tfrac{1}{\sqrt{3}}\sin 2x = 0\) na intervalu \([0,\,2\pi]\)? Zatim razmotrite: ako bi koeficijent pred \(\sin 2x\) bio \(-\tfrac{1}{\sqrt{3}}\) umesto \(+\tfrac{1}{\sqrt{3}}\), koliko rešenja bi postojalo na \((0,\pi)\)?

Pogledaj nagoveštaj

Interval \([0, 2\pi]\): iz \(\sin x = 0\) dobijamo \(\{0,\pi,2\pi\}\) (tri rešenja). Iz \(\cos x = -\tfrac{\sqrt{3}}{2}\) u \([0,2\pi]\): \(x = \tfrac{5\pi}{6}\) i \(x = \tfrac{7\pi}{6}\) (dva rešenja). Ukupno \(3 + 2 = 5\) rešenja.

Koeficijent \(-\tfrac{1}{\sqrt{3}}\): sada je \(\cos x = +\tfrac{\sqrt{3}}{2}\), čije rešenje u \((0,\pi)\) je \(x = \tfrac{\pi}{6}\). Slučaj \(\sin x = 0\) i dalje ne daje ništa. Ukupno opet \(1\) rešenje, ali drugačije locirano.
